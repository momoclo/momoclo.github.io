ももクリ 2016 Day 1

ももクリ2016 Day1 に行ってきたよ。
演出などのネタバレをさけつつ、軽くレポートします。

■会場・チケット
幕張メッセ 4/5/6 ホール
3 ホール連結 (幕張メッセで連結できる最大規模) でセンターステージ
今年のコンセプト通り、暖房の設定温度は 28 ℃だそうですｗ
→ ひとが多いので暑かった

海浜幕張駅の周辺で何かボードをもったひとが結構いたので見てみると…
→ 「ミュージックステーション同行させてください」のひとが多かった
→ 今日はメッセで他にも色々なイベントをやっている模様

チケットは自力では全部はずれ
→ 知り合いの赤推しさんに連番させてもらいました どうもありがとう
→ 指定席 A5 ブロック 通路最前 (ブロック最後列)
ゴンドラが来たらチャンスかも？と思ったらあんまり来なかったｗ
→ 一度だけ通りましたが、メンバーのほとんどは外周方面を向いていて内側をあまり見てもらえず

会場内はフラットで、モニターもセンターステージの上に 4 面のものがあるだけだったので、遠くのひとや角度が悪いひとは全然見えなかったらしい
→ 自分のところからもステージは ほとんど見えず、モニターを見ている時間のほうが長かった

■お天気
朝から風が強く、気温も高くてなんだか 12 月という感じじゃない天候
海浜幕張駅から幕張メッセに向かう途中、少しだけ雨がぱらついてましたが基本曇り
→ 帰りも晴れており、寒くなかった

■てまきパーク
事前認証と物販のために立ち寄りましたが、導線がひどいことに
さらに、パンフレットを販売している場所がいつも (最近だとはるえ商店) と違い、桃黒堂書店で別扱い
→ タイミング悪く LV の入れ替えでロックアウトされてしまったので買えませんでした
→ 現場だと荷物になるし、Day 2 は てまきパークでスタンディング LV なので通販で買いました

■開演前
入場待機列の案内の手際が悪く、客入れに時間がかかったので 5 分押し
→ ツイッターで状況がわかったので、特に困るようなことはなかった

17:00 影ナレーション開始
有安さんのところだけメモ
も「今日はクリスマスという、キキョウな日を…」
→ 貴重ですね、わかります
→ キキョウを噛みつつ、何度か言い直していたけれど結局 最後は「大切な日を…」とかいう風に言い換えてましたｗ

他にも「一期一会」とか、噛みそうなワードが適度にちりばめられてました
→ たぶんこういう台本を読まされていたと思いますｗ

アナウンスの声は風邪をひいているのか、だいぶ鼻声っぽい調子
→ 今日の本番のコンディションはまったく問題がなかったので、事前に収録した時に風邪気味だったのかな？

■音響
大きなホールで締め切り、という条件でしたがエコーはひどくなかった
→ ドームツアーではエコーが結構きいていて聞き取りにくいこともあったけれど、今日はそんなことはなかった

■タイムテーブル
17:05 開演
19:30 本編終了
19:35 アンコール開演
20:20 終演

■有安さん
白い風の気張りやすの評判がよかったです
→ 飲み会で他推しのひとからたくさん言われました
アンコールの時はツインテールだったよね？

■最後の挨拶
有安さんのところだけメモ
も「今日は久しぶりのセンターステージで、どこ見てもどこ見てもどこ見ても」
→ 「どこ見ても」を 3 回繰り返してました
も「モノノフさんがいる しあわせを感じました」
も「ここは暑いけれど、外は寒いので風邪をひかないように」
も「素敵なクリスマスイブのイブをありがとう」

この後、かなこちゃんが挨拶をしていた時に…
か「今日の幕張メッセはたくさんイベントがあって、色々なお客さんがいるので」
→ ここで有安さんがファインプレー
も「帰り道で、大きな声で『マジももクロのライブたのしかったー！』って話して帰ってね」
→ 電車で乗り合わせたミュージックステーションの観客なんかにアピールをする作戦のようです

■まとめ
このところソロコンサートや、5 人揃わないイベントが多かったので、やっぱり 5 人だなー！とあらためて感じました
セットリストも新旧織り交ぜつつ、あまり休憩や MC を挟まずにコンパクトに圧縮して進めるような展開だったので、どことなく懐かしさを感じるライブでした
ココ☆ナツでサークルになって楽しそうに踊るももクロちゃんや、冬の定番曲の数々などなど、とっても楽しかったです
