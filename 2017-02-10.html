ももクロ裏伝説

川上アキラのももクロ裏伝説 公開収録に行ってきたよ。

■会場
センテニアルホール
→ 横浜アリーナのホール
→ 古くは 2012 年春の一大の LV 会場だったところ
→ 最近は物販会場だったり、認証会場だったりでおなじみの場所

■タイムテーブル
19:00 開演〜ゲスト：吉田豪さん
19:30 ゲスト追加：ハッスルプレス 井上さん (以降 3 人でトーク)
20:40 終演

■開演
川上さんが登場してすぐ、私立恵比寿中学 松野さんについてコメント
→ コメント形式に書き起こせるほどキチンと覚えていないのと、内容が内容なのでうまく書けないけれど、ももクロのマネージャとしての立場からと、スターダストの幹部としてお悔やみを言っていました

川「ももクロのメンバーたちは、明日のライブに向けて、今日もレッスンをしています」
川「みなさんを笑顔にできるよう、明日もがんばりますので、どうぞよろしくお願いします」

■トークショー
オンエア前なので…、というかほとんど ももクロに関係ない話だったので割愛
吉田さんはそこそこ ももクロの話が出たけど、井上さんは大半 関係ない話をしていた
→ 全体のボリュームとしても、川上さんよりもしゃべっていたんじゃないか？と思うほど

■有安さん
お誕生日に向けて、デジタルデータではなく形に残るものを、ということでバースデーブックを製作中
→ ハッスルプレスから川上さんに初稿チェック依頼をステージ上で念押し
ハ「月曜日までにお願いします」

今回、リレー形式でメンバー対談を行う企画があり、有安さんは次にお誕生日を迎える玉井さんと対談
→ ブログかなにかで「あるメンバーと長時間対談して緊張した」とか言ってたね

玉井さんもソロコンサートをやったら？という話題で…
も「絶対ソロコンサートをやったほうがいいよ！」
し「わたしは縁の下の力持ちタイプなんで、やりたくない」
も「それなら、わたしがプロデュースするよ！」
→ 実現するかどうかは不明ですが、楽しみにしておきます

■質問コーナー
ゲストが降壇してから、川上さんに質問する時間がありました
モノノフ「ソロコンのディスク化はありますか？」
川「今年中には出せるよう、色々と考えています」
→ おおおおお！？
→ 質問はいくつか出たけど、コレが一番盛り上がりましたｗ
川「一人一枚で出す予定」
→ 全部まとめて、という形ではなくそれぞれでディスク化してくれるらしい
川「ただ、そうすると初めの頃の高城さんのは映像化を考えていなかったこともあって、撮れ高が少ない」

■まとめ
ももクロ裏伝説というタイトルの割に、ももクロの話が出ませんでした…
