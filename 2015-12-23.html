ももいろクリスマス2015 Day1

ももいろクリスマス2015 Day1に行ってきたよ。
二日目以降にまたやりそうな演出は伏せる形で軽くレポート。

■シャトルバス
10:30からのチケットを買っていたので、時間にバスのりばへ
少し行列ができていたので乗車・出発が11時前くらい
軽井沢スノーパークに到着したのが11:50

■音声ガイド
シャトルバスの車中では ももクロちゃんの音声ガイドという名の放送あり
出発してすぐに開始…するんだけど、最初の5分くらいはどうでもいいことを話していて注意事項的な話にならないｗ
5分くらいして注意事項をアナウンス
も「き、キキョウ品の扱いにご注意ください」
→貴重品のももかつぜつで車内に笑いが起きてましたｗ
放送は移動中ずっと流れるくらいの尺があり、軽井沢スノーパークに到着してぴったりに内容終了
→ バスの運転手さんに「放送とぴったりの到着ですね！」と声をかけたら「そうですね?」となんだか嬉しそうでした

■チケット
横浜の紫推しさんが連番してくれました どうもありがとう
Cブロック 整理番号後ろ
13:10 に入場待機列に並び、14:00 に会場入り

■開演前
BGM は定番ウィンターソング
ロマンスの神様
恋人がサンタクロース
スノーマジックファンタジーなどなど

■開演
14:35 開始
セットリストに関係しそうなのは伏せときます

■音響・お天気
序盤はマイクの音量が小さかったかな？
→ ももクロちゃんたちの声も最初は出ていなかったかも
→ 序盤以降はだいぶ改善されてました
天気予報では午後から雨の予想でしたが、ライブ中は曇りでした
気温は1度くらいだったそうですが、それなりに防寒対策をしていったため体感温度では寒くなかった

■ステージ
モニター　ステージ
B　　　A
D　　　C
というブロック配置
ステージが右よりで、左側にモニターがひとつある感じ

Cブロック後方からはステージの高さが低く、今年の女川復興祭と同じくらいでほとんど見えなかった
モニターの大きさは大宰府政庁跡の男祭りと同じくらいのサイズかな？

■ももかちゃん
衣装はネタバレになりそうなので省略
髪型は触覚＋ストレートだったと思うけれど、あんまり見えなかった
→ なので表情に関するレポートは少なめ
カメラに向かって、とあるアピールをする演出があるんだけど、そのときに投げキッスもしてたよね？
→ 詳しくは演出にかかわるので以下同文

■MC
「寒いのでどんどんいくよー」
「MCも今日は少なめで！」
ということで、MC 自体すくなかったけれどいくつか小ネタを紹介

■れにちゃん自己紹介
れ「ゲレンデマジック、高城れにです！」
れ「今日はいつもよりも一段とかわいい (はーと)」
→ ゲレンデマジックですね、わかります

■ジャンプ
「今日はジャンプしてもオッケー」
→ アップを始めるかのごとく、ぴょんぴょん飛びはじめるモノノフのみなさん
「なんだかモグラ叩きみたい」
→ 観客席後方からもそう見えましたｗ

■リアルスモーク
吐く息が白くなっている…という話の流れで
「みんな、上に向かって は?っ てしてみて」
→ 一斉に「は?っ」とするモノノフのみなさんｗ
「本物のスモークみたいになってるよ！」
「特効いらないね」
→ 特効の代わりにやるのは大変そうですが…ｗ

■雪
アンコール最後の曲、さらに曲の終盤で頭上から雪がちらほら
→ 最初は人工雪の特効かと思いましたが、リアル雪でした
雪に気がついたももクロちゃん、テンションがあがって声が上ずってましたｗ
「雪が降ってる！」
「初日からコレでハードルがあがったね」
「明日はどうなるかなー」

■有安さん語録 アンコールで登場後
も「みんながアンコールをしているのを見て、凄い綺麗だなーと思いました」
→ 多少演出に関係するのでややぼかしてますが、アンコールしているところをしっかり見ていたそうです

■最後の挨拶
も「みんな、こんなところまで来てくれるなんて、ももクロのこと好きでしょう？」
→ はーい！
も「わたしも、こんなところまで来てくれるモノノフさんのことを、さらに好きになりました」
も「最後の雪は演出かと思いました」
も「雪が降っているのをみて、有安さん感動しちゃいました」
→ 正確な言い回しは例によって適当ですが、自分のことを「有安さん」と言っていたのは合ってるはず
も「むにゃむにゃむにゃ…」
→ なんか言ってたけど、ももかつぜつ気味でヒアリング難度高めｗ
も「し、舌がまわらなくなっちゃった」
→ かなこちゃんにパス

か「ももクリの詳細を発表したとき、会場がスキー場というのを聞いた…」
か「みんなのなんとも言えない顔が忘れられないですが」
→ 確かに、お、おう…という反応だったよねｗ
か「今日一日、楽しんでもらえたと思います！」
→ はーい！

か「最後に雪が降ってきたときには、ももクロ持ってるなーと思いました！」
→ 今までにも台風が来そうだったのが晴れたりと、何度も ももクロちゃんお天気伝説を目の当たりにしてきましたが…
→ ホワイトクリスマスの演出を呼びこんじゃうミラクルにも立ち会うことができました

か「それでは、最後にぜーっと！で締めたいと思います」
あ「みんな、もうすぐ帰れるからね」
→ 念のため補足しておくと、寒いのを気遣っての発言ね
→ ぜーっと！で締めて終了

■川上アキラのここだけの話 in 軽井沢スノーパーク
終演後、規制退場でC、Dブロックが規制解除待ちをしていたら…
川「みなさん規制退場にご協力いただき、どうもありがとうございます」
川「せっかくなので、残っているみなさんにだけいくつかお話をします」
川「まずAブロックが完全に退場してからね、Aブロックは早く帰れー」
→ Bブロックが退場後、Aブロックが出ていく途中でした
→ ちょっと大げさに表現していますｗ

■ももクロちゃんと電話
既にバックステージ？ホテル？に戻った ももクロちゃんに川上さんが電話
全部は覚えていませんが、いくつか紹介

あ「今日の雪は、あたしが雨女なのでできた演出です」
あ「ということで、最高の演出家だったでしょー？」

も「有安だよー」
川「え、なんだって？」
も「有安だよー」
→ 「だよー」の部分が何故かヒアリング難度高めｗ
川「だよーまで滑舌わるいんかーい」
川「まだ帰れないで残っているひとに暖かい話を」
→ と無茶振りしたところ…
も「ないよ！」
→ 即答でしたｗ
川「会場の有安さんファンのひとー？」
→ はーい！
川「えーと、ペンライトが28本あがってますね」
→ この手のくだりだと、たいてい適当な数字 (しかもいつもかなり少なめ) を言う川上プロデューサー
→ ちなみに しおりんの時にも 緊急推し人数調査 in 軽井沢スノーパーク が行われて…
川「えーと、さっきの有安さんよりも3本多いですね、29 本」
→ ももクロちゃんたちから「さっき28本って言ってただろ！」とツッコミが入ってましたｗ
川「じゃあ31本」

■規制退場
こんなやりとりを聞きつつ、Cブロックが会場を出たのが18:00ちょい前くらい
そこからバスのりば方面へ移動＆行列をしてバスに乗ったのが18:55
軽井沢駅に到着したのが19:40
シャトルバス乗り場から、駅までは5?10分程度かかります
その日のうちに電車で移動するひとの参考になるかな？

■まとめ
思ったよりも寒くなくて、感覚としては去年？の 紅白歌合戦 横浜赤レンガ パブリックビューイング と同じくらいの服装で大丈夫かなー？という感じでした。
近くに居合わせたひと (Beautiful Survivors) から、チョコレートをもらったり、飴ちゃんをもらったりしながら楽しく見てたけど、表現の仕方によっては「冬山で遭難したひと同士で食料を分けあっている」のとあんまり変わらないですねｗ
