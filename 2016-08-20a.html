ロケハン vol.7 青森 1 部に行ってきたよ。

■会場
青森県 五所川原市 ふるさと交流圏民センター オルテンシア
キャパシティ 1,100 程度
開演時間が 12:30 で、東京から始発の新幹線に乗っても 12:08 に五所川原駅 着 → さらに駅から徒歩だと 20 分
→ 始めは土曜日の始発で行こうと思いましたが、厳しそうなので金曜日の夜から前乗りティー
話題には出ませんでしたが、オルテンシアはイタリア語で紫陽花という意味らしい

■のっけ丼
青森駅近くに宿泊したので、土曜日の朝に 青森魚菜センター 古川市場へ
有安さんがポンコツロケで食べることの出来なかった、のっけ丼をいただいてきました
朝 7:30 くらいだったためか、モノノフの数はそれほど多くなかった

■タイムテーブル
12:30 開演
13:55 終演

桃神祭のテーマ からの?、和楽器アレンジの桃神祭バージョン Overture で開幕
→ 舞台袖からペンライトがぴょこぴょこはみ出てましたｗ
→ ももクロちゃんが振っていた模様

■Long Time No See
「青森って何時ぶり？」
→ 地元民に質問するスタイルｗ
→ 5 年前にイオンに来たとか、あーりんやす？推され隊？でムニャムニャ (時期とか場所とか忘れた) に来たという証言

■最高齢ノフは誰だ in 青森オルテンシア
一番年上のモノノフは誰？という話の流れに
川「50 ？」
→ はーい！
川「もっと上いる？」
→ 55! 60! 61! などと、なんだかセリをしているような雰囲気にｗ

男性では 61 歳？の方がその場にいた最高齢？ということでステージに上がったんだけど…

も「ちょっと、あそこに お婆さんがいるよ！」
→ ステージから緑推しの白髪の女性を見つけてました
川「ステージまで来られますかね？」
白「足が悪いので…」
→ すかさず ももクロちゃんたちがステージから降りて、お婆さんの元に駆け寄ってました
まわりのノフが座席を譲り、ももクロちゃんがそこに座ってお婆さんと記念撮影

このお婆さん、横浜アリーナの有安さんソロコンサートのときに偶然見かけていて、「親類の方かな？」と思っていたら青森まで追っかけるガチ勢だった模様ｗ
→ 終演後、お話ししたら「行けるところには行きますのよ」とのことでした すごいね

しばらくほったらかしだったステージの男性は れにちゃん寄りの推され隊推しとのことで、事務所にもっと推されたいをリクエスト
→ステージに椅子を用意してもらい、まさしく特等席で見てました
→ あーりんは観客席で鑑賞

■箱の中身はなんじゃろな
っていうか、いつもの質問を書いたやけに深いボックスですが…

→ iPad mini をボックスから取り出す

■feel a heartbeat
ボックスからリクエストが飛び出る
あ「あたし白幕やるね！」
→ 横浜アリーナで登場するときにステージにかかっていた幕の役をやるらしいｗ
れにちゃんが有安さん役で幕の向こう側にスタンバイ
有安さんは一度ステージ袖に

イントロがかかり、幕がはらはらひらりと落ちると…
→ れにちゃんが歌詞をすっごくあやふやな調子で歌う
有安さん、なかなか出てこないなと思ったら、スケッチブックにフライパンの絵を描いて登場
フライパンをジャカジャカしながら歌っていたので、ギターということが判明
も「今度、ギターの絵を練習しとくね」

サインを入れたフライパン風ギターの絵は即席ジャンケン大会で勝者のノフさんに進呈
も「売ったりしないでね (・Θ・)」

■あーりん無双
有安画伯のイラスト争奪ジャンケン大会で、有安さん、れにちゃん、あーりんとジャンケン
→ 有安さんで勝ち抜け状態のノフが半分になり…
→ れにちゃんでまた半分になり…
残り数人というところで、ボスあーりんが登場
→ あーりんが一度に全員に勝ってしまい、全滅してましたｗ

■あーりん教育
リクエストボックスに入ってましたｗ
→ れにちゃんが入れたのかな？

→ 音源がなかなか出なくて、川「また後でね」となっちゃいました
→ 結局、その後は忘れられててやらなかった

■のっけ丼
も「前に青森に来た時に食べられなかったー！」
「たべたひと？」
→ はーい！
→ 今日たべたよ！
も「今日やってるの？」
も「明日の朝、食べて帰ろうかな」
→ 今晩じゃなくて、明日帰るのかな？

スケジュールについては、れにちゃんが「今朝、東京は雨が降ってた」とも言っていたので、当日の朝移動だったみたい
→ 2 部で あーりんが「飛行機でごはんを食べた」とも言っていたので、彩高コンビ＋川上さんは朝のフライトで来たんだね

有安さんは当日の始発の新幹線で東京 → 新青森、そこからクルマで会場入りしたそうです
