幕が上がる 上映会

幕が上がる 上映会に行ってきたよ。

■ブールセック
埼玉方面でイベントがある度に寄っている気がする
今日も苺のタルトをいただきました
12:00 頃に行ったら、相変わらずの混雑ぶりでした
→ 今日は店員さんがいっぱいいて、対応にぬかりがないといった雰囲気

■会場
キラリ☆ふじみ
埼玉県富士見市の市民文化会館
有安さんが子供の頃、ダンスの練習や発表会でステージに上がったことがあるらしい

■タイムテーブル
13:30 - 14:00 待機列形成
14:00 - 14:15 入場〜富士見市紹介ムービー#1
14:15 - 16:15 映画本編
16:15 - 16:20 富士見市紹介ムービー#2
16:20 - 16:50 トークショー

700-800 人ほど並んでいて、15分で入場大丈夫かな？と思っていたら待機列のままどんどこ入場して間に合いましたｗ

■平田オリザさん
最初に登場
キラリ☆ふじみの舞台責任者？をやっていたり、長年かかわっているそうです
軽く自己紹介をしてから有安さんを呼び込み

■有安さん
登場すると会場はペンライトで緑に
オ「地元の富士見市の方、ドン引きですみません」
衣装はドームツアーでよく見るニッカボッカ風のアレ
髪型はツインテール
オリザさん＋有安さん＋川上さんでリラックスしているのか、テンション高め
挨拶のときに富士見市トークで、ブールセックのことも紹介してました

■川上さん
オ「実はもうひとり、富士見市にゆかりのある方がゲストで来ています」
→ 川上さん登場
川「どうも御存じない方は、このデブ誰だろう？といったところでしょうが宜しくお願いします」
川「今日は富士見ローカルのディープな話しかしません、よそから来たひとは切り捨てで」

■富士見市とふじみ野市
似ているけど別物という意識が強いらしく、今日は富士見市のイベントなのでふじみ野市の話題はご法度らしいｗ
オ「富士見市とふじみ野市は仲悪いので」
オ「NHKよりもNGワードが多いイベント」
オ「(質問コーナーでふじみ野市の話題にならないか) こんなに緊張する質疑はないよね」

■オリザさんと有安さん
映画の打ち上げのときに、富士見市の話題で共通点があるんだなと認識
→ 有安さんが子供の頃はふじみ野市はまだ無かった (合併前) ので、正確にはどっちの市に住んでいたのかあやふや
→ その場で有安ママに電話をかけて、どっちだったか確認したそうです
さらに、有安さんがキラリ☆ふじみに出ていた頃から舞台にかかわっていたので、きっと前に会ってますよねといった話

■質問コーナー
富士見市の方を優先して質問を受け付けることに
会場の質問者のところへは川上さんがマイクを持っていく係
も「結構はしれるんで、あのデブ(・Θ・)」
男性#1「富士見市で景色がいいなと思う場所はどこですか？」
も「えー、どこだろう…」
も「間違えてふじみ野市の場所を挙げちゃうとマズイ」
も「どっちかわかんないんですけど、親水公園の桜が綺麗だと思います！」
→ 地元の方の反応が「正解！」だったので、無事に富士見市のスポットだった模様

女性#1「自分も川上さんと同じ幼稚園でしたー！」
→ それだけ言いたかったらしいｗ
他にも同じ小学校 (鶴瀬西小学校) だったひとも名乗り出てたよね

女性#2「有安さんと同じ登校班だった、ムニャムニャじゅんぺいの母です！」
→ 有安さん、意外な方の出現にビックリしつつご挨拶ｗ
#2「小学生のころから、ダンスが凄くて、手足がピーンとしていて」
→ やたら褒められていたので有安さんがステージをバタバタ走り回って照れてました
→ かわいいー！と囃し立てたくなるドタバタっぷりｗ
#2「最近はケーキ屋さんに緑の格好のファンの方がいっぱい来ていらして」
#2「愛されているんだなーと思っていつも見ています！」

■地元トーク
川「大と書いてビッグと読むお店が鶴瀬駅にあって〜」
→ 焼き鳥屋さん？か何かの話、地元の方は結構喜んでましたｗ
有安さんと川上さんが髪を切っていたお店が同じらしい
→ トマトという美容室？
川「二か月にいっぺんくらい、よそに引っ越ししてからも通ってました」
も「オーディションの前とかに寄って髪を綺麗にしてもらってました」

■委嘱式
映画の上映会の前にあったPR大使委嘱式の話題
→ こちらはアマランサス衣装で出ていたそうです
同席したミスユニバース？の女性が背の高い方々だったらしく…
川「有安さんの倍くらい身長のある方で」
オ「でも年齢は同じくらいなんでしょ？」
も「年齢は訊かなかったのでわかりません…れにちゃんみたいに「歳いくつー？」といけないので」

■次回作？
どんな流れだったか忘れたけれど…
も「わたしは背が低くて、滑舌が悪いので…」
オ「それじゃ、背が低くて滑舌が悪い役で書きます」
→ 次回作を書くなら？ってことだったっけ…？

■コール＆レスポンス
観客の誰かが「ももか好きー！」という内容を叫んだら、それに反応して
も「わたしもみんなのことが大好きー！」

「ももかー！」と掛け声がたくさんかかる中、「有安さーん！」との声にも反応して
も「なーにー？」
→ 夏の西武ドームの「なんだようー？」を思い出しました

■記念撮影
報道用の撮影と、エンジェルアイズ用の撮影と、ブログ用のスマートフォン撮影
も「ももいろクローバー」
→ ぜーっと！
スマートフォン撮影は観客が一度に入りきらなかったので、左右にわけて撮影
→ 自分は下手側写真の、有安さんのすぐそばに写っているはず

■最後にひとこと
も「今度は歌で、富士見市を盛り上げられたらと思います」
も「次は歌を歌いたいです」
川「では来年 (の委嘱式で) はここを会場にしてイベントを」
川「オリザさんが舞台責任者をしていたくらい、音響がよいし」
川「ステージから観客席の一番後ろまでよく見えるいい会場」
オ「では来年は富士見市民限定で、ここでやりましょう」
オ「みなさん今のうちに住民票を移しておいてくださいね」

平田さん、川上さんが退場して、一番最後に有安さんが去り際に…
も「みんなー、7/３のソロコン来てねー！」

■HMV
ららぽーと富士見店にも寄ってきたそうです
浦和美園店からレンタルしていたサインを返すために、富士見店むけに新しくサインポロシャツを進呈
他にもポスターにサイン＋メッセージを書いた紙もありました

■出待ち
終了後、10分たらずで平田さん、川上さんと一緒に出てきました
私服に着替えていて、白黒のスタジャン姿
しばらくモノノフに手を振ったりしてから、川上さんのポルシェケイマンに乗って帰っていきました
→ 後部座席にひとりで乗っていて、窓を両側とも開けてどちらにも手を振ってくれました
