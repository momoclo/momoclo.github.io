Halloween Party 2015

Halloween Party 2015 にも行ってきたよ。
メモは避難所参照。

■会場
幕張メッセ 9/10/11 ホール
お台場のライブ終了後に移動して 16:00 頃 海浜幕張着
少しお店に寄ってから 16:40 くらいに入場列へ

■観客
去年と同じく、仮装率はかなり高め 仮装じゃないひとを探すのが難しいくらい
モノノフ以外のファンは化粧なれしている女性が多く、メイクも本格的でした
モノノフもいつもとは趣向が違ってハロウィン要素を入れているひとが結構いました
16:50 頃にホール内へ

■開演前
ホール内へ入ると、開演前なのに既に真っ暗
暗さに目が慣れる or 携帯電話などでライトをつけないと座席の位置がよくわからないレベル
自分は指定席を希望して、希望通り指定席 80 列
→ アリーナの最後方ブロックでしたが、ブロック内では最前＆すぐ前に大きなモニターがあり、ステージを直接見るのは厳しかったですがパフォーマンスはよく見えました
→ モニターはかなり充実しており、大きめのが全部で 5 つ
→ 設置位置も高めだったので、どの席からもよく見えたと思います
17:05 開幕

■Breakerz
DAIGO「アリーナ盛り上がってるかー」
→ いえーい！
D「あ、今日はアリーナだけですね…」

D「今日は氣志團さん、ももクロさんとバラエティー豊かな方も出ますんで」
→ というより、バラエティー班的な扱いをされていた気がしますｗ
D「ダイゴォ～↑↑っての、やってもらえます？」
→ ダイゴォ～↑↑
D「めっちゃテンション上がるね」

■Xishidan
登場時にはモニターに大きく X JAPAN みたいなロゴが
→ よーく見ると、X じゃなくて K らしい (X の左側に縦線が入っているような感じ)
→ そして大きな X の下に ishidan の文字

最近の X JAPAN がライブでどんな格好なのかは知らないけれど、今日の氣志團ちゃんは紅とかを歌っていた頃の X さんと見分けがつかないくらいのメイク＆髪型＆衣装でした
→ 演奏＋歌のパフォーマンスもかなりハイレベルでした
翔「すいません、出オチです」
翔「もうこれ以上、広げられないんで」
と言ってましたが、モノノフには馴染みのある喧嘩上等、定番の One Night Carnivalといったセットリストやスローモーション演出など、楽しかったです

■ももクロちゃん
19:00 メインアクトだと Breakerz、Xishidan さんの後に登場
さっそく小芝居から開始する安定の展開

か → アリス 水色のワンピース
し → チェシャ猫 ピンク＋猫ヒゲ (マジック)
れ → マッドハッター 黄色のタキシード
も → ウサギ シスターラビッツ＋赤いちゃんちゃんこっぽい羽織
あ → ハートの女王様 映画の Alice in Wonderland にそのまま出てきてたようなデザインの赤いドレス

…という具合にカラーがいつもと違っていたので、隣の Vamps さんファンの女性 2 人組はかなり混乱している様子でしたｗ
→ あれ、センターの赤い子ってどれ？？？みたいな感じ

■小芝居
不思議の国のアリス
流れはもうこのスレッドに出ているので省略


アリスの不思議の国茶番。不思議の国で変わった時計ウサギ(ポンキッキ杏果)を見つけて追う夏菜子アリス。
終われて怖がり泣きべそウサギ。
どうしたの？と来る玉チェシャ猫とマッドハッターれに。
アリス「ちょっと興味があっただけなのに」

「どれどれ(ウサを)、ホントだ、これは汚れた大人になってしまった自分にしか興味がなくなってしまった『元子役』だ！」
杏果「そんな腹黒とかエゴサしかできないとか、うぇーん泣」
夏「自分で言ったら否定できないよ」
ここでハートの女王あーりんかっこよく登場。
あ「どうした？」

あ「なるほど、子役が成長して脱皮のタイミングに苦しむよくあるやつだな。安心しろ、私もおはきっずからここまでになった。おっはーー！！」
夏「ポンキッキーズとおはきっずがいるグループはうちだけだ！こんな脱皮の仕方もあるんだってよく見とけーー！」
曲開始。


→ 「ポンキッキーズと おはキッズがいるグループはウチだけだー！」で盛り上がるモノノフのみなさんｗ

■自己紹介
いつもバージョン
れ「鋼少女」
し → 「しからないで～」と言い始めるも、やっぱり間違えたのでやり直し
し「最近、自己紹介を変えてまして…間違えて古いのをやっちゃったのでもう一度やります！」
し → しゃべるの大好き
お → おしゃべり大好き
り → 理由はないけど
ん → んー？
し「泣き虫で甘えん坊は卒業、ももクロの若大将、玉井詩織です！」
→ まだ迷走中っぽいけど、ももクノかタカハシでやったのと同じバージョンでした

セットリストはかなり変化球でした
→ フェスで初めて見たひとはついていくのが大変だったかも
→ コールやミックスの少ない曲が多く、一体感を感じてもらえたかなー？

冒頭の小芝居と自己紹介以外、その他は特に MC などは無かったかな？
19:40 頃に出番終了

■Halloween Junky Orchestra
Junkie のほうが一般的な気もするけど、公式サイトを見たら Junky ですね
21:00 スタート
ステージに今日の出演者が勢揃いし、Hyde さんが指揮者よろしくタクトで采配しつつハロウィンのテーマソングを演奏
→ タクトの振りにうまく合わせられなかったりするとアウトー！な感じで、演者さんには緊張が走ってました
→ 有安さんはトライアングル担当
→ タクトが左右にぶんぶん振られる度にキョロキョロ追っかけてて、マタタビに反応するネコみたいになってましたｗ

去年と同様、「アフリカの子どもたちがよくやるリズムゲーム」コーナー
ルールは文字だと説明するのが大変なので省略 見れば一目瞭然なんだけどね
全般的に ももクロちゃんはこのゲームが苦手？というか慣れていないせいか、アウトー！と思われる場面がちょくちょくありました
→ 割と判定が甘め＆審判さんに贔屓してもらっていた感じで、れにちゃん以外バツゲームにならなかった
→ かなこちゃんもアウトー！な感じの時がありましたが Hyde さんがバツゲーム
→ ももかちゃんは完全にアウトー！でしたが、「ももクロのファンが多い」という謎理由で隣のひとがバツゲームになるミラクル判定ｗ
→ 「なんでオレなの…」と納得がいかない様子でしたｗ

■バツゲーム
「箱の中身はなんじゃろな at 幕張メッセ 2015」← 命名は適当です
→ 以前は あかりんがいた頃から、かなり無茶なお題でやったりしてたよねー
→ 緑推しとしては「うどん昨日たべたぁ」を期待してしまうよねｗ

■ももかちゃん #1
まず最初に練習でやってみたところ、いきなりアウトーｗ
有安さん、顔がマジになってますｗ
→ でも練習だったのでバツゲームはナシ、緑推しとしては残念？だけどセーフ

■れにちゃん
本番一発目でさっそくアウトーｗ
高城さん、顔がマジになってますｗ
→ やっちまったーという文字が顔に描いてありそう、挙動不審にバツゲームの位置へ
中身は「白滝」
→ 最初は恐る恐るという感じでしたが、一本をつまみ上げると…
→ その後は片手でぐわっと掴み上げるなど、結構思い切りがよかった
れ「わかった！白滝？！」
→ 正解！

■かなこちゃん
隣の Hyde さんとのコンビネーションがうまくいかず、どっちがアウトだろう？という微妙なところ
→ ふたりとも「ヤバイ…」という表情をしていましたが、Hyde さんがアウトの判定
→ Hyde さんはウナギでした

■ももかちゃん #2
完全にアウトー！というところでしたが、前述のとおりミラクル判定で隣の方がバツゲーム
→ 有安さん、「すすすすいません」という感じに謝ってました (ジェスチャーを確認)
→ 中身は「カエル」
→ アマガエルとかのかわいい感じのではなく、結構大きいガマガエルっぽいのが 2 匹ｗ
→ しかも 1 匹はやけに威勢がよく、手のところまで伸び上がっていたりしてました
モニターにはバツゲームの様子しか映ってなかったので、自分は双眼鏡で有安さんを観察してました
→ それでも遠かったので表情はあまりわからなかったけど、モニタのカエルの様子を見てかなりビビってましたｗ

■終幕
ゲーム＋ゲスト紹介なんかで 40 分くらいやってた気がする
その後 20 分くらいはハロウィンソングを延々とやりつつ、Trick or Treat でお菓子をばらまいてました
最前ブロックにいた方にゲットできたか訊きましたが、お菓子が軽いせいか投げても全然飛ばず、最前列近くじゃないとゲットできなかったらしい
22:00 少し過ぎたところで閉幕 最後のアナウンスで時計を確認したら 22:03 でした
