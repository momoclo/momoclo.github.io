AYAKA-NATION 2016

AYAKA-NATION 2016 に行ってきたよ。

■会場・観客
横浜アーリナ
→ あーりんブログではこう記述されていたけど、公式サイトは「横浜アリーナ」
→ あーりんが正義！ということで「アーリナ」を個人的に採用

今日は当然のことながら、ピンクのひとだらけでした
→ 自分もピンクのシャツで参加
→ 知り合いの緑推しの方とお会いしましたが、お互いピンク装備だったので「いつもと違うね〜」
→ その方は全身すべてピンク＆奥さんまでピンクのツナギ姿だったので、知らないひとが見たら林家ペー・パー子さんと間違えそうｗ

■HRC Yokohama
午前中に事前認証を済ませて、みなとみらいのハードロックカフェ横浜まで行ってみることに
ロックショップのウィンドウディスプレイはあーりんバージョン
→ あーりん桃神祭衣装
→ ハスラー仕様あーりんロボ
→ ももいろパンチ衣装＋ガムテープで補強していたブーツ
→ ミニあーりんロボ＋あーりんパペット
→ フォーク村あーりんギター

昨日の氣志團万博の疲れを癒やしつつ、近くでから揚げプレートを食べて横浜アーリナへ

■デイリーヤマザキ
横浜アーリナそばのデイリーヤマザキでは「綾鷹」をイチ推し
→ 「今日はコレでしょ！あやたか」と宣伝ポップつき
→ 自分もつい買ってしまいましたｗ

■開場
16:30 に開場、現場は小雨模様
傘をさしているひとが多く、列がどうなっているのか見通しがよくわからない
→ それでも、スタッフに整理されるわけでもなくきちんと つづら折りになって待機列ができてました

■開演前
ステージはハートマークのワープ装置と A をかたどった階段

→ ひみつのアッコちゃん
→ キャンディキャンディ
→ 花の子ルンルン
→ 魔法使いサリー
などなど

■あーりん講座
衣装チェンジのときに流れた手描きアニメ？
→ 描いては消し、描いては消し…のあの手法、なんて言うんだろ？

とっても面白かったですが、ボリュームが半端なかったので、キーワードを列挙
→ 亜林大学あーりん学部あーりん学科 (大学名以外は覚えていません：適当)
→ 霊長類 ピンク妖精属 あーりん目
→ TOAIC - Test of Ahrin for International Communication - 国際コミュニケーションあーりんテスト
→ 954 点？で 2 位の成績
→ 1 位は「あーりん＝かわいい」の法則を発見した アーリニスト・A・アーリーン教授 (適当)
→ 世界各地には様々な種のあーりんが生息
→ その種類は 156 種類
→ イチコロあーりん で語呂合わせをして覚えてね！
→ (細かいのも入れると？) 9156 種類 (適当)
→ 悪魔もイチコロあーりん
→ 手長あーりん
→ 手長足長あーりん
→ マウンテンあーりん ← ゴリラかよ！
→ 日本でおなじみなのは、日本あーりん・あーりん
→ 野生あーりんを捕獲するには、カゴの下に から揚げの罠をしかける (古典)
→ あーりんに遭遇したら逃げちゃダメだ 追いかけてくるぞ
→ 口裂けオンナの伝説は、そんな あーりんの生態がモデルという説も
→ そういう場合は落ち着いて「あーりんかわいい」と褒めること

■あーりんの"素"

■衣装・髪型

途中、ハートマークのワープ装置を使った高速衣装替え
→ 赤白のどことなくケーキ風？衣装からレインボードレス
→ 紙吹雪で青い衣装に一瞬で変わるなど、見ごたえのある内容

中盤はホワイトのウェディングドレス？＋髪をアップにまとめた姿

ロックパートは黒い革ジャン？＋ワインレッドのギター
→ その後は白いシースルーのスカート追加

アンコールは物販シャツをデコレーションしたもの
あ「みんなと同じの着てきちゃった」
あ「自分でいうのもなんだけど、あーりんがいっぱいいて怖い」
→ シャツの全面に大きくあーりんがプリントされてましたね

■青春賦
歌詞の内容をそのまま手話でも表現
→ 自分は「ありがとう」と「ごめんなさい」の手話しか知りませんが、歌詞にも「ごめんね ありがとう」があるので、そこで手話だ！と確信しました
→ あーりんが直接その意図を話すことはありませんでしたが、色々なひとに伝えたい ということかなー？と解釈をして泣きました

■終わりの挨拶
ランウェイをまたいでカメラが移動できるように、通路を作っていたんだけど…
あ「そこ繋げてもらっていい？」
→ 繋げてサブステージに行きたい
→ ジャンプ！という声が挙がりましたが、あ「無理無理！」
→ 上から見ていると割と距離があったので、走っても衣装やなんかで確かに無理そうでした

あ「セットリストとか演出で頭が一杯で、MC で何を話すか考えてなかった…」
→ という通り、あーりんにしては珍しく言葉に詰まったり、お水を飲みに行く回数が多かった
→ その分、その場で思ったことを素直に話してくれた気がします

あ「あーりんの髪、長いのと短いのどっちが好き？」
あ「帰りたくないなー！」
あ「みんなが あーりんを好きでいてくれるのと」
あ「その 2 倍、あーりんはみんなのことが好きだからね！」

■まとめ
フルコースの名の通り、キュート＆ロック＆セクシーなあーりんが見られる内容
それに加え、爆笑できる要素もふんだんにある、とても楽しいエンターテイメントでした
れにちゃん・有安さん・ももたまい・あーりんとイベントをやってきたけれど、各メンバーそれぞれの個性が出て面白いですね
ムービーで出てきた、「エンターティナー」や「史上最強アイドル」という称号がよく似合う、ナチュラルボーンアイドルのショーであっという間の 3 時間でした
新曲の歌詞にもあったけれど、「あーりん is forever!」
