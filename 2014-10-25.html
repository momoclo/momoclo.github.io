東京ハロウィンパーティ

東京ハロウィンパーティに行ってきたよ。

■会場
パレード：お台場のフジテレビ近くの散歩道
ライブ：フジテレビ前のイベントスペース (野外)
自分はパレード会場に12:30頃到着 スタート／ゴール地点のすぐそば 最前列

■パレード
12:30頃は最前列にいけたものの、それから急にひとが増えてきて12:50頃には結構な人だかり
既に一般参加のパレード列は列を作って待機中
12:55頃、フジテレビ三宅アナウンサー、加藤アナウンサー(だっけ)が登場
そのすぐ後にアーティストの仮装グループがきて、ももクロちゃんも登場

パレード列は他にも仮装者やスタッフが多く、小さいももクロちゃんは隠れちゃってやや見えない状態
→ ももクロちゃんからも観客がよく見えないらしく、顔に「(モノノフさんが) 見えなーい」と書いてある気がしたｗ

■仮装
かなこちゃん → ミニーマウス
しおりん → ラプンツェル 茶髪っぽいカツラ
ももかちゃん → ティンカーベル 金髪のカツラ
あーりん → 佐藤錦 (サクランボ)
れにちゃん → アメリカンチェリー (サクランボ)

自分の目の前3メートルくらいをゆっくり歩いていたので、まわりからは推しの名前を叫ぶ歓声が上がりまくってた

あーりんとれにちゃんはお揃いのサクランボ着ぐるみ
「5人で1つの衣装を着たいです。」(by あーりん) の事前コメント通り、くっついてた
帽子がヘタになっており、葉っぱのところで2つがくっついた作り

kwkmさんはいつもの格好＋天狗のお面
三宅アナウンサーはキョンシー
加藤アナウンサーはエルサ (アナと雪の女王)

か「今日は好きな衣装を着てきたので、(統一性がなく) バラバラです！」

三宅アナウンサー「しおりんは全然黄色関係ないですね」
→ ラプンツェルはピンクっぽい衣装

れあ「山形(←ユニット名の模様)です！」
れあ「ひとつになりたい！」

■パレード
10分ほど、スタート地点でインタビューやオープニングセレモニーをした後13:05頃パレードスタート
パレードコースは結構な距離があったので、最前を狙えば結構いい写真が撮れたのかな？
→ あとで追っかけたひとの話を聞いたところ、ずっと人だかりが絶えなかったそうです
自分はスタート地点でゴールを待つことに
13:40頃、ゴールに戻ってきた気配がして期待していたら、ゴールは別コースらしくスタート地点は肩透かしに
→ みんなゴールのほうに大移動していた

仮装に参加したひとはアーティストの後にスタートする形になったので、一般の観覧客ほどはももクロちゃんを見ることができなかったかも

■ライブ前 待機列
自分は0312番 15:15組
15:00頃 集合場所に到着すると既にひとがいっぱいおり、待機列を作り始めてた
自分の聞こえる範囲では800番くらいまでいたかな？
観客はいつものモノノフ格好のひとがほとんどだけど、ハロウィンにちなんだコスプレのひともちらほら

待機列からはフェスなんかのPA調整の「ツェー、テストテスト」という声が聴こえる
15:40頃、ようやく入場開始

■ライブ会場
フジテレビ前のイベントスペース ちょっとしたフードコートっぽい場所の一角
関係者エリアが大きく取られていて、スタッフや他アーティストさんが見学していた
→ 関係者にも ももクログッズのタオルを持ったひとがいたｗ

入場時に紙製のメガネが配られる

自分はブロック2番目の最前列 上手側
すぐ後ろが背の低い女性2人組だったので、フェンスに顔を乗せる形で低めの体制

後ろの女性2人組「Overtureでウリャオイやってると笑われちゃうよね…」
→ 最近どこかのフェスに行ってアウェイな雰囲気を感じることが多かったようです

■開演
少し押して16:05頃スタート
三宅アナウンサー、加藤アナウンサーがステージに
「みなさん、今日は先ほどお配りしたメガネをかけてくださいね」
→ 観客を映したのを見る限り、ほとんどのひとがメガネをかけてた

軽く挨拶をしてから、三宅アナウンサーがペンライトと推しタオルを取り出す
→ ペンライトをひとつ加藤アナウンサーに手渡し、タオルを首にかけて…
→ 「戦闘態勢が整いました！ももいろクローバーZの登場です！」
