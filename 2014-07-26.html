Summer Dive 2014 Nissan Day1

日産スタジアムに行ってきたよ。

■座席・音響
バックスタンド(E23)ほぼ最上段。スポットライト機材のすぐ下あたり
場所のせいか、音響が今まで参加したライブで一番悪かった
たまーに風に乗って音が聞こえてくる感じ
まわりのコールのほうが大きく、歌声も演奏もそれに負けるほどのボリューム
すぐ近くのカップルが歌の間もMC中もひっきりなしにしゃべっていて集中できなかった
ステージに対してのリアクションやコールではなく、男がやたらと実況・解説をしていた
「ほら見て、ホニャララ〜」とか「あれは前にムニャムニャ…」とかとか

そんなわけで内容は薄め。
さらに今日のステージセットや演出、ゲストについてはネタバレになりそうなので伏せます。

■開演前
「パンチ」と書かれたTシャツを着たももクロちゃんが やまだうどんのパンチの宣伝をする VTR が流れてた
「わたしたち、今あえるアイドル！週末ヒロイン ももいろクローバー、、、パーンチ！」

■ステージセット
例の5億円のアレ
遠目にはドリフのコントのセットと大差ないように見えたｗ
→ 5がつく数字はファンタジーだからね、うん (50人のおじさんとか)
…けれど、近くだったひといわく「結構豪勢な作りだった」

■新曲
My Dear Fellow に振り付けがついたのでダンス披露
→ 野球モチーフの動作が多く、歌詞とあいまっていい感じ
→ 最後にサブステージからスタンドに向けて5人それぞれでボールを投げ込んでた (1球だけ)
→ そこそこ飛んでいるように見えた…少なくとも極楽門の軍手よりは飛んでた
→ 着地点はよく見えなかったけど、スタンドに届いたのかな？

MOON PRIDE
5人のチームダンスがかわいくかっこよく、こちらもとてもいい感じ
セーラームーンによく合った振り付けだと思った

月虹
振り付けは地味め
なんとなーく、月と銀紙飛行船っぽいなと思った (振りも含めて全体の雰囲気が)

■ももかちゃん
MC中に突然ドリンク補給に
「え、そのタイミング？」と言われるくらいに唐突
「有安水」と書かれた例のアレをモニターにちゃーんと映る向きで持ってストローでチュー
「あたしケチだから、普通の水いれるかもしれない」

ココ☆ナツの放水ではカメラの前にアクリルの板を置いてガードしたところ目がけて発射
…するつもりが操作にてまどり、なんだかモタモタｗ

歌の真っ最中にカキ氷を食べたり、屋台めぐりをしたりｗ
→ 他のメンバーもだけど、ところどころ歌詞が飛んでたよね

「5年前は、こんなことになる (日産スタジアムでライブ) とは思わなかった」
→ もちろん加入した時のことですよねー

メンバーにいじられまくり、最近よくテレビで見る ふくれっつらに
→ 少し口をとがらせてほっぺをぷー
→ モニターに大きく映っていたので、これはよく見れた

ゲストさんと滑舌対決
→ 有安さんの勝ち…だと思う

有安企画
ソロコンサート！？と思ったらそうでもなかった
→ ももかちゃんがサプライズ発表担当に
→ ちびっこ祭と女祭り
→ 男祭りの発表はなく、そのあたりはなんだかウヤムヤに…

何かの話の流れで日産スタジアムがグリーン一色に
メンバー「国立競技場に続いてここでも！？」
→ なんとなく流れ的に Words of the mind ~brandnew journey~ が来るかなーと期待したけれどそんなことはありませんでしたｗ

ランウェイでチビッ子ダンサーと一緒に踊る
→ どっちも似たような衣装
→ 遠目だと区別がつきにくく、曲が終わってランウェイから降りて捌けて行ってしまったので、あれれ？と思ったらそっちはチビッ子ダンサーだったｗ

自分の席の近くの女性2人組 年齢はももかちゃんとほぼ同じか、ちょっと上(20代前半)くらいかな？
→ 緑推し「うちのこが今日で加入5周年…」
→ 同年代に「うちのこ」と言われてしまう有安さんって一体…

終盤 曲が終わり、サブステージ側からセンターステージに移動する時に…
アリーナ側、スタンド側の観客に向けて手をぶんぶん振りまくる
→ それだけでなく、ランウェイにいたサポートダンサーともハイタッチをしたり色々
→ センターステージから距離が遠かったのもあるし、足が遅…もとい、ゆっくり歩いていたのに加えてそんな調子で愛想をふりまきまくっていたらステージに戻るのが超遅くなってた

ももかちゃん「スタンド最上段のみんなー、たのしかったですかー？」
スタンド最上段で音響が小さく、まわりのノイズもあって今いち乗り切れなかった今日のライブだけど、この一言で撃ち抜かれました
放水はぜぇんぜん届かなかったけど、最後にレーザービームがずどーんと来るとは…
