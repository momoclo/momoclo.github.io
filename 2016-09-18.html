氣志團万博 2016

氣志團万博に行ってきたよ。

■会場
袖ヶ浦海浜公園
今年は 2 ステージ制で、交互にやるタイムテーブル
→ ももクロ前のサブステージ側の何組は見られなかった
→ メインステージ張り込みだと、インターバルが 40 分と長い

足場が沼地になっている心配をしていたんだけど、それほどでもなかった
→ 芝生は割と乾いており、そのまま座れた
→ 最前の柵は足元が金属なので、こちらも問題なし＆最前近くもそれほど泥になってなかった
→ 10-FEET のダイブエリアがやばそうでした

■観客
例年どおり、フェスらしく多種多様な人種が入り乱れ
→ 自分はステージセンターからやや下手側、最前
→ 10-FEET のファンが少し残ってた (下がれない？) けど、まわりはモノノフだらけ
今年は矢沢さんのファンが新顔で、キッシーズとはまた違った意味で怖そうな方々がチラホラｗ

■お天気
天気予報では、お昼前後に少し雨が降るはずだったけれど、結局降らなかった
→ たまーにポツポツとあったけど、傘は全然いらない程度
一日中曇り空で、風が強かったので過ごしやすかった
→ 都内の自宅に帰宅したときは小雨でした ももクロちゃんお天気伝説…！

■行き
10 時くらいに東京方面から向かう京葉線は、ディズニー＋東京ゲームショウ＋氣志團万博のお客さんで混み混み

■タイムテーブル
自分は 11:00 頃に到着、エビ中のステージを見てからメインステージへ
→ 最前で ももクロまで見たあとは、メインとサブを行き来しつつフードエリアの LED も

ももクロちゃん
15:55 - 16:35 予定通り、きっちり時間いっぱいでした

■ももクロちゃんセットアップ
次が ももクロちゃんの時はスタッフさんがステージにバミリをたくさんしてました
バンドセットなどはないものの、謎の小道具？が用意される
→ かなこちゃんの飛び道具でした

サウンドチェックではステージ裏からドラムを叩いている音が…
→ ローディーさんのチェックという感じではなく、やや微妙なテクニック＆荒ぶり具合
→ これは後ろで有安さんが荒ぶっているのでは！？と妄想してましたが、正解はどうだったんだろ？

■アーティスト紹介ムービー
翔「もうね、房総半島は ももクロ半島でもいいと思うんですよ」
翔「毎年、袖ケ浦市長が挨拶に来てくれるんですけど、ボクと握手しながら ももクロの楽屋のほうずっと見てるんです」
翔「ももクロちゃんはどこだ！？ってね」
翔「少なくとも、袖ケ浦市は ももクロ市でいいと思います」
翔「今年も、ももクロ万博おねがいします！」
ナレーション(立木さん)「房総のアイドル、暴走…！」

イメージイラストはジャングル大帝 (以前にもあったのとほぼ同じ)

■ももクロちゃん参上
5 人そろってステージへ
→ 整列したところで、くるりとあっちを向いてスクリーンへ
→ スクリーンには大きな日の丸
川「それでは、国歌斉唱を行います」
川「みなさま、ご起立ねがいます」
→ スタンディングエリアは当然たっていたけど、フードエリアで座っていたひとたちはどうだったんだろ？
君が代を全員で斉唱
…からの Overture で ももクロ万博2016が開幕

■おっぱいミサイル
開幕早々、かなこちゃんが胸に白いロケット風船を装備ｗ
→ それを有安さんが空気入れでキュコキュコ
→ ロケット風船が順調に伸びていき…
→ ミサイル発射！？と思ったら ばーんと破裂してましたｗ

■自己紹介
ショートではなく、いつもバージョン

■氣志團カバー
今年も愛羅武勇
→ 両手挙げて合図しろ〜♪のあたりの振り付けは去年？で覚えたので楽しかったです

■ツヨクツヨク
タオルは安定の逆さま
→ We are the mononofu と、ちょっとどっちが上かわかりにくいから間違い
→ …とも言えなくもないけど、きっと正しく出したら「(いつもと) 逆！」と言っちゃうよねｗ

■ザ・ゴールデン・ヒストリー
ジャンプのあたりでは下手側に来てくれました
→ 飛ぶときに脚がコンパスみたいに広がっているのが可愛い（・Θ・）
怪盗少女のフェイクでは下手側に来なかった分、このときに下手側にアピールしてました
→ ステージの端っこのほうでちょっとバランスを崩したときに、ゲロッパみたいになってた
→ 最近、だいぶオトナになったなーと思ってましたが、そうでもなかったようですｗ

■行くぜっ！怪盗少女
今年の新記録チャレンジは大縄跳びｗ
→ ステージで大縄跳びをしながら歌うアイドルって他にいないんじゃないかな…ｗ
あーりんが中々はいれなかったり、入っても ひっかけちゃったりしまくってました
→ かなこちゃんはエビぞりジャンプから戻ってきて、そのまま走った勢いですんなり入ってました
→ 普通、何度かタイミングを図ると思うんだけど、さすがだなー

本番前の練習ではどうやっても長続きせず、本番が一番うまくいったそうです
→ 夢と希望の世界新記録達成 (だっけ？)

■ピンク空
ラストはピンク空
→ 残念ながら、夕暮れには少し早い＆曇り空だったので、ピンク空にはならなかった
でも今年発表の曲でセットリストを構成するあたりは新鮮でよかったです

■有安さん
今日はストレートヘア
ブログに上がるまで気がつかなかったけど、確かに少し長さが短くなってました
→ 以前の れにちゃんくらいかな？

■衣装
衣装はアイドルらしいスカート
今年の桃神祭？こんなの着てたかなー？と謎でしたが新作らしい (川上さんのツイート)
→ 風が強かったのですこしめくれ気味でした

■実は紫推し？
ももクロの出番が終わり、軽く休憩しようとステージから離れると…
→ 紫推しの方々が誰かと記念撮影してました
→ 誰だろ？と思って近くのモノノフさんに訊いたら、れにちゃんのパパママでした
パパさんは噂通り、コワモテでしたが写真撮影にきっちり応えてくれるなどイイ人そう
ママさんはもっとぶっ飛んだ方かと思ってましたが、そうでもなく美人さんでした

以前、れにちゃんが「ウチのパパは赤推しなのー」とかなんとか言っていた気がしますが、パパママさん揃って紫装備でした
→ 上半身はユニフォームと帽子がパープル、靴も紫だったような…

■氣志團コラボ
夜 20 時くらい
氣志團のステージ中…
翔「2012 年に氣志團万博をやる、という時に真っ先に手を挙げてくれたアイドルです」
翔「この 5 人がいなかったら、氣志團万博は開催できませんでした」
と紹介されて登場

衣装はセーラー服 (夏服でスカーフがメンバーカラー)
→ れにちゃんだけ長スカートのスケ番風、他は膝丈くらい
→ こちらも強風でめくれ（・Θ・）ばきゅーん
コレだけでも充分かわいかったですが、サンタ帽をかぶっていて天使？妖精？という感じでした
