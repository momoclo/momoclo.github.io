COUNT DOWN TV GIRLS FES

COUNT DOWN TV GIRLS FES 2017 に行ってきたよ。

■会場・チケット
代々木第一体育館
COUNT DOWN TV の公開収録で、チケットは無料
→ 開場前から、スタッフさんが「途中退場はできません」とアナウンスしていました
チケット自体は SOLD OUT になったみたいですが、タダだったせいか実際に来場するひとが少なく…
→ アリーナは 6-7 割の埋まり具合
→ スタンドは 3-4 割くらい？
→ 2F スタンドの観客を全員 1F スタンドに移動させてちょうど埋まる程度

■ゆるキャラ
開場して中にはいると、d point のイメージキャラのポインコ兄弟がいました
→ 一緒に写真を撮ろうと思ったけど、ひとが多かったので断念
その先には、COUNT DOWN TV の 3 人組もいました

■開演前
2F スタンドに座って待っていると、18:30 頃にスタッフさんが来てブロックごとに 1F スタンドへ移動するように指示
→ ブロック単位で 1F スタンドに行き、空いているところに適当に座りました
→ 自分はピンクのシャツを着て行きましたが、右隣はパリピっぽい大学生くらいの男性 3 人組 (Flower ファン)
→ 左隣は Little Glee Monster ファンの 20 代カップル (Little Glee Monster が終わったら帰っちゃった)
まわりにはそこそこモノノフがいたけど、全体としては 3-4 割くらいかな？

■タイムテーブル
18:40 開演
22:00 終演
あーりんの出番は 20:55 - 21:20
一度バックステージにもどったあと、すぐに抽選に参加するために再登場 21:20 - 21:30

■あーりん参上
20:55 NEXT ARTIST のトレイラーで登場
→ 最初はマイクの音量がおかしいんじゃないかと思うくらい、あーりんの声がでかかったｗ
→ 結局、あーりんの出番中ずっとそんなだったので、機材調整ミスじゃなくてあーりんの声がでかいという結論

Overture の時こそ戸惑っていた様子でしたが、その後は「あーりん！あーりん！」と超ノリノリでしたｗ

■抽選
出演者のステージ転換など、合間合間にライブを終えたアーティストがクジをひく抽選会
→ タオル賞 x 5
→ d point 賞 (サイン入りグッズ) x 1
→ ナントカ賞 (アーティストに会える権利) x 5 組 (2 x 5 の 10 人で、あーりんに会えるのは無かった)

他のアーティストは割と淡々とクジをひいて「南スタンド 2F、F ブロック 3 列 15 番」とか発表するだけだったんだけど、あーりんはこういうのに慣れているせいか、観客と会話のキャッチボールをして進める

→ おおおお！？

→ おおおおおお！？
といった感じ
