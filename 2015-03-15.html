舞台挨拶ファイナル

舞台挨拶ファイナル ライブビューイングに行ってきたよ。
5 人全部書いていたら超長くなるし、そもそもそこまで覚えていないので有安さんだけレポート。

■会場
TOHOシネマズ日本橋 スクリーン7 F-15
これで鑑賞 9 回目 (「その前に」は 2 回)
→ 同じ映画を 10 回程度見るのは自分にとってそれほど特別ではないです

■開演前
18:28 バルト9 の様子が中継されはじめる
観客の顔がわかるレベルで映ってた
→ 「あーりん卒業おめでとう」
→ 緑のハッピーバースデーケーキ型の帽子をかぶっている二人組
→ 20 歳おめでとうのメッセージ＆カエル帽をかぶったももかちゃんのイラスト

■開演
18:35 スタート 司会はフジテレビ三宅アナウンサー
三「ももクロのみなさんを BGM なしで、スタンディングオベーションで迎えましょう」
→ かなこちゃん、しおりん、れにちゃん、あーりんが先にステージ上へ

ももかちゃんは今日がお誕生日なので、ハッピバースデー♪でお迎えすることに
→ 全員でハッピーバースデーを歌う中、ももかちゃん登場
→ 自分は LV だけど歌ってきました

■ロウソク
れにちゃんのメラメラ芸ではなく、観客のペンライトを吹き消す演出
ももかちゃんは背伸び、というか小さくジャンプして客席のペンライトをふー
も「絶対ヘンな顔してるんで、下から撮らないでください」
→ プレスのひとにお願いしてましたｗ
→ 自分は LV だけど吹き消してもらいました

■舞台挨拶完走について
三「ももかちゃんは完走と、今日お誕生日なのでそれについても」
ももかちゃん、色々と長く話してくれたけれどちょっとした事件があったので内容は忘れました

も「全国あちこちの劇場に舞台挨拶にいくと、どこでも老若男女 (← 言えてない)、様々なモノノフさんがいて〜」
→ 老若男女は 2 回くらい言い直してましたｗ

も「あのー、ムニャムニャで〜、あのー、ホニャララが〜、あのー、ゴニョゴニョは〜、あのー」
…といった具合に「あのー」が少し多め
→ 会場の雰囲気が「あのー」をいう度に笑いが起こるやや謎な空気に

自分はそれほど気にならなかったし、「あのー」をいくら言おうが、ゆっくり自分の言葉で考えてしゃべってくれれば充分なので、落ち着いて待っていたいところだったんだけど…

途中で「あのー」が多いことで会場に笑いが起きていることに気がついて、そこからは少しギクシャクしたコメントになっちゃっていた気がする
→ お誕生日についても最初は「特にないです」と急いで自分のコメントを終わらせようとしていたよね？

LV では映っていなかったけれど、ももかちゃんのコメント中にペンライトをぐるぐる「巻き」の指示を出しているひともいたのかな？
→ クノの wikipedia で調べてきたことを発表するときにぐるぐるはわかるけれど、どうなんだろうね

■20 歳の抱負
その後、あらためて三宅さんが振ってくれたので、20 歳になった抱負を語る
→ この時も「あのー」を気をつけるあまり、自然にしゃべるのに苦労していたように見えた

も「1 年 365 日もあるのに、去年の国立など、自分の誕生日の時には色々と節目のイベントがあって神様がそうしているのかなと思います」
→ さらに言えば、舞台挨拶がスタートした 18:30 が生まれた時刻ですね

最終的には「あのーと言って話が長くならないようにしたいと思います！」とまとめてた

■ミッション
かなこちゃん → 宣伝部長
しおりん → 司会力向上
あーりん → 時代小説 (なんだっけ？) を読む → 全 12 冊のうち、6 冊まで読んだそうです
ももかちゃん → モノノフの笑顔 (原文ママ) を写真に撮る

…と、メンバーそれぞれにミッションが課せられていて…
→ 4 人でがんばったよねーと健闘を讃えあう横で、れにちゃんが複雑そうな表情ｗ

れ「あたしは特別なミッションがなかったけれど、仲間に入れてよー」
→ その後も一番 舞台挨拶の回数が少ないなどいじられてましたｗ
も「でもソロコンサートをやったのは凄いと思う！」
→ 推され隊から素晴らしいフォローがあったよね

■ムロさん
電話で出演
ムロさんも舞台の千秋楽だったらしく、妙なハイテンションで噛みあわない会話をしてましたｗ

■オリザさん
手紙を三宅アナウンサーが代読
「今日は舞台の脚本の締め切り日なので、そちらに行くことができません」
「舞台挨拶には自分も参加させてもらいましたが、どこにいっても歓迎してくれて、ももクロのみなさんはいいファンに囲まれていると思います」
→ モノノフが褒められている時に喜んでくれるももクロちゃんたち
「みなさんは、どこまでも行ける切符を持っています」
「自分の仕事が、宇宙の果てまで行ける銀河鉄道の、停車駅のひとつとなっていたら光栄です」

…といった内容の素晴らしいコメントをいただきました

■写真撮影
いつも通り、ももかちゃんが客席＋4 人を撮影
→ 掛け声なんだったっけ？
も「声が小さい！もっと大きな声出せるでしょ！」
→ ムニャムニャー！
→ すかさず映り具合をチェック
も「(プレスさんの) フラッシュなしでもう一度」
→ プレスさんがももかちゃんに向けてフラッシュを焚いていたので逆光？になったようです
→ その後、無事に撮りなおして終了

■南ピー
そろそろ終わり、というところで「愛のメモリー」が流れる
→ おおおお！？
→ 南ピーさんではなく、プロデューサーさん 3 名が客席後方から登場
→ 替え歌を歌うはずだったものの、緊張して歌を飛ばしてしまったらしい

モリヤP「みなさん、あちこちに舞台挨拶に行って大変だったでしょう」
→ ももかちゃん、首を左右にぶんぶん振ってました
れ「舞台挨拶をやめたくなーい！」
P「やりたくないということで…」
→ それだと意味が逆だから！と突っ込みを入れるももクロちゃんｗ
P「やめたくないということで、kwkm さんと急遽相談して 4/11 にもどこかで舞台挨拶をやってもらうことにしました」
→ まだ具体的なことはまったく決まっておらず、詳細は後日発表とのこと
→ 4/11 だと TAKAHASHI の前日ですねー、忙しいと思うけどがんばってね！

■終演
サプライズ発表もあったせいか、予定時間を大幅に押して終了
れにちゃん、しおりん、かなこちゃん、あーりん、本広監督が先にバックステージへ
ももかちゃんが退場する間際に…
→ 観客「ももいろクローバー」
→ も「ぜーっと！」
と決めポーズをやって去っていきました
最後に三宅アナウンサーが退場して終わり 19:35

■まとめ
しゃべることを考えていたり、「あのー」をいくら連発していても、ゆっくりももかちゃんが話すのを聞いていたいなーと思いました。
舞台挨拶は観客との距離が近い分、そういうのがちょっと難しかったかもしれないね。
必要以上に心配するのもアレだし、余計な詮索をするのも何なので、ちょっと気がかりではあるけれど、自分はこれからも静かに応援していこうと思います。

■写真
ピンクハウスも撮影許可をいただきました
