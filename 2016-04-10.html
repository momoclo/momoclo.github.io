Pretty Guardians

Pretty Guardians 発足記念 Fan Club イベントに行ってきたよ。

■キラリ☆ふじみ
朝 7:00 に到着、いたって平和に待機列形成
抽選整理券は 25 番をゲット、配布総数は 394 だったらしい
その後も列を崩すことなく 8:00 まで待機
チケット購入番号 整理券は 90 番をゲット
→ 整理番号 160 くらいのチケットを購入できました
結局、9:00 に並んでいたひとは全員チケットを買えたらしい (この時点で残り 100)
その後の書き込みを見ると、12:39 残り 90 枚、15:00 残り 70 枚というペース

■HMV ららぽーと富士見
チケット購入後にサインを見に行きました
すぐ目の前に見えているけれど、横断歩道の都合で結構あるく
さらに、ららぽーとの一番奥にあるため、入ってからも結構あるく
→ ポロシャツに有安さんのサイン
→ 2015-02-24 浦和美園の舞台挨拶時に、HMV 浦和美園店向けに書いたものを展示しているらしい

■ブールセック
ふじみ野駅からのイオンシネマ大井で舞台挨拶時に語った、思い出のケーキ屋さん
自分は埼玉方面に出かけるついでに訪問している (先週の西武ドームの時にも寄りました) ので、これで 5 回目くらい
10:30 頃に到着したら、既に先客が 5-6 名ｗ
→ 苺のタルトを取り置きしておいてもらったので、さくっと食べて退店 (レジ待ちで時間かかりました)
その後も続々と緑っぽい格好の方が来店し、15 分ほどの滞在で 20 人くらいかな？

■セーラームーンイベント
12:30 頃に到着

■会場・観客
東京体育館
まぎらわしいけれど、代々木第一体育館・代々木第二体育館とは別
国立競技場の横にある、10,000 席ほどのアリーナ
→ 入った感じだと、横浜アリーナよりも少し小さい代々木第一体育館といったところ

観客はノフ率だと 1-2 割くらい？
男女比は 2:8 くらい？
全部で 2,200 人くらいで、スタンドは使用せずにアリーナのみでした
→ 2 割弱くらい男性がいて、男性はほぼノフかな？という印象
→ 女性はたまーにノフがいるけれど、反対にほぼセーラームーンファンっぽい

■タイムテーブル
14:55 - 15:00 オープニング？主題歌ムービー初披露
15:00 - 15:30 開演〜野島さん・三石さん・小佐野さんトークショー
15:30 - 15:50 28 話先行上映
15:50 - 16:10 野島さんライブ・堀江さんライブ

16:20 - 16:55 プレゼント抽選会・ムーンライト伝説合唱・月に代わってお仕置きよ！〜終演

■ももクロちゃん登場
堀江さんのライブが終わり、「次はももクロちゃんの出番です」という具合に呼ばれる
Overture
MOON PRIDE
自己紹介 (ショートバージョン、名前と年齢を言うだけ)
月虹

野島さん・堀江さんのライブ時は全員着席して歓声＆拍手 (ペンライトは ほんの少しだけ)
ももクロちゃん登場時からはノフがポツポツと立ち上がってペンライト (といっても 1-2 割)

冒頭、かなこちゃんが「ニュームーンに恋して」の話題を出そうとして…
か「こ、恋…」
し「ニュームーンに恋して」
→ すかさず訂正してましたｗ

■本日の有安さん
ツインテール 位置は若大将フェスティバルよりも若干上だった気がする
衣装は豪華版のセーラームーンコスプレ

■プレゼント抽選会
チケットの半券を全出演者がロトマキノ的に箱から取り出す方式
セーラームーン関連のグッズや、声優さんの私物のレアグッズなど、結構豪華
→ あたった人数は全部で 50 人くらいかな？
本来は「A3 ブロックの 15 番」といった具合に、ブロックと番号をセットで読み上げないと特定できないんだけど、最初のうちはブロック番号を読まずに「15 番！」といった調子だったのでみんな？？？とモヤモヤしてました
→ 途中から直った
→ 最後に、ブロックを読み忘れた分については改めて読み上げ直した

■抽選係
当選番号をボックスからひく係＆読み上げる係は全出演者が交代
ももクロちゃんたちは、5 人それぞれで担当
→ 有安さんは隣にいた堀江さんとアイコンタクトを取りながらボックスをひいてました

■ジュピターと同じグリーン担当のあの方
プレゼントされるセーラームーングッズについて詳しく、ボックスをひくときに小ネタをちょいちょい披露

も「(グッズ A について) これは私も買いました、まだ 2 個しか集めてないんですけど」
→ あかしれ「(そこの緑の子は) ファンですから！」

も「(グッズ B について) これ、なかなか手に入らないんですよね…わたしも まだ持ってません」
→ あかしれ「ファンですから！」

も「(グッズ C について) これ、光るバージョンのもあるの？光らないのなら持ってる（・Θ・）」
→ あかしれ「ファンですから！」
→ セーラームーンの小道具の形をした USB の充電バッテリーだったかな？

そういえば、番号を読み上げる時に…
も「A1 ブロックの、148 番！」
→ あかしれ「ひゃく よんじゅう はちばん！」
→ といった風に、残りのメンバーが復唱してました
→ 堀江さん「5 人いて (フォローしあってて) いいわね〜」
→ ももかつぜつをフォローしていたのかな？
→ 途中からは恒例になって、ももかちゃん以外の読み上げでも ももクロちゃんたちが復唱してた

ボックスから半券をひくときも、手袋をしたままだったのでちょっと苦戦してたね
も「手袋がツルツル（・Θ・）」

■ムーンライト伝説
最後に全出演者で合唱
「みんなも歌ってねー」
→ と言われたものの、スクリーンに歌詞が出たわけではないのでノフ的にはシドロモドロｗ
→ 自分は 1 番だけ歌詞を覚えていたので歌いましたがかなり小声でした
→ ファンの方々は歌詞を覚えていそうだけど、あんまり声を出していなかったので、全体としては観客は歌わず、出演者の合唱を聴いている状態

■締めの挨拶
全出演者が挨拶、ももクロからは代表かなこちゃん
か「わたしたちも、セーラームーンのファンとして、演者として、これからも応援します！」
か「みんなも応援してくれますか？」
→ はーい！ (会場は全体的に静かで、返事をしていたのはノフさんがポツポツといった感じ)

■決め台詞
最後にあの決め台詞
三石さん「これからも、セーラームーンを応援してね」
三「応援してくれないと…」
三「月に代わって」
→ お仕置きよ！

事前に「お仕置きよ！」を全員で＆大きな声で、とリクエストがありましたが会場はかなりおとなしかった
→ 調子に乗ってポーズ付きでやっちゃいましたが、まわりは全然やってなかったｗ

■まとめ
ドームツアー全通時のどの席よりも、今日の席のほうがステージに近かったです。
久々にアウェイ現場を体験しましたｗ
Lady Gaga のオープニングアクトの時に、リトルモンスターだらけのブロックに紛れちゃったときと同じ気分
→ あの時は会場のノリ自体はよかったので、今日の大人しい会場のほうが色々と肩身が狭い感じだったかなー？
そういう雰囲気を味わうのも含め、結構たのしかったです。
