Girls' Factory 2015 Day1

Girls' Factory Day1 に行ってきたよ。

■会場・座席
今年も代々木第一体育館
トートバッグ＋うちわをくれるのも同じ
自分は17:40過ぎに到着
→ 少なくともこの時間では原宿駅側の門が解放されていて、裏までぐるっとまわることはなかった
→ 終演後も原宿側の門から駅方面に出られました
→ 去年は終演後もぐるっと裏側をまわったような…

チケットはヤフー先行だったかな？
南スタンド2F Gブロック 上から2列目
→ 今まで一度もアリーナ・北スタンドになったことがなく、毎回かならず下手側スタンド

■観客
カラフルなひとだらけだけど、ももクロのライブとは少し雰囲気が違う…という感じ
座席はすべて埋まっていました

■モニタ
メインステージにあったモニタは奥にひっこんだ状態なので、G ブロックあたりの横からは見えませんでした
サブモニタがステージ脇にあったものの、インタビュー映像と MC の時に画面が使われただけで、歌パートのときはずっと GF のロゴがぐるぐる回るのみで、実質無いのと同じ
→ 今日も双眼鏡を持って行ったので、双眼鏡で見ていることが多かった

■開演前
17:45 頃に入場したら、既にインタビュー映像が流れてました
例年通り、控えスタジオ？で冬美ちゃんをインタビューする超綺麗なお姉さんが…
→ 入場してすぐ、遠くから見たときは玉井さんのはず、と思っていても「本当に？」と思うくらい印象が違ってた
→ 髪が肩までの長さで、衣装もすごく落ち着いた感じ
→ 白い生地に小さ目のドット模様のワンピース＋ヒールの高いサンダル
やっぱり髪型は重要だなあ、と思いました

17:55 に照明が暗くなり、音楽が流れだして「始まる？」という雰囲気
→ テレビに合わせて 18:00 ちょうどにスタートしたのかな？

■ねごと
しゃちほこと共演 → ねごしゃち
エビ中と共演 → ねご中
…の流れで てんかすトリオとコラボ → ねごかす
いきなり唯一の持ち曲、永遠のトリニティーをやっちゃいました
有安さんがセンターで歌う構図
→ バンド編成だったので、本当にガールズバンドのボーカルぽく見えた
→ てんかす 3 人とも緑のラフな衣装
→ ももかちゃんは髪を結ばずにストレート

■玉井さん
「ももクロの若大将、玉井詩織です！」
「今日はひとりなので、落ち着きませんが…」と言いつつも大分なれた感じの進行
エビ中かしゃちほこの曲が好きという話題で…
→ 歌ってー！
「4 時間で 40 曲やるから、時間がないよ！」
「わたしは今日 MC だから！」
→ でも最後みんなで歌うところでは玉井さんソロパートもあったよね

■エビ中インタビュー
し「今年の目標は？」
エ「紅白に出ることです！」
し「スタダ枠が無くなる…、ももクロも紅白 4 回目に出られるように頑張ります！一緒に出ようね」

■バックバンド
今日のもダウンタウンももクロバンドということでいいのかな？
→ コーラスやリズムギターなどはいつものメンバー
アレンジのせいかもしれないけれど、今日の演奏はよかったと思います
→ リードギターのひとがももクロバンドのひとじゃなかったよね？
→ 西川さんでも、佐藤さんでもなかったはず
→ ステージ下手側、黒いストラト＋ナチュラル テレキャス＋赤いアコギのひとの演奏がよかった

■たこやきレインボー
さすがに認知度低いんじゃ？と思ったら、会場のほとんどがフリコピやコールを完璧にしてました

■チームよつば
し「グリーンの 4 人ですけど、どう？」
も「実はみんなで集まるの、まだ 2 回目なのであんまり仲良くない（・Θ・）」
→ 歌を歌った後に…
し「これをきっかけに仲良くなってね」

■シークレットゲスト
冬美ちゃんが紫の和服で登場
冬「あの方をお借りしてもいいかしら？」
し「どうぞどうぞ」
冬「今日の衣装はこの色なので…」
→ れにちゃん登場、予想外？だったので会場が盛り上がる
し「高城がシークレットゲストで登場して、会場が沸いて悔しい」

■カラオケで練習した成果
れ「幕が上がるの舞台で、27 回も勝手に歌った曲があって…」
冬「勝手にと言わず、どんどん歌ってね」
→ れにちゃんと冬美ちゃんでデュエット
冬美ちゃんはさすが演歌歌手という歌いっぷり、会場がおおおおと感嘆してました

■紫しきぶ
れにちゃんがセンター
し「高城さんがセンターですよ！」

■てんかすトリオ再び
ねごかすで出番が終わりかと思ったら、終盤がっつり出てくれました
衣装はモノトーンのメイドさん風
有安さんは背中部分がほぼ全面白など、黒よりも白のほうが多い印象
髪型はさっきと変わり、ポニーテールに

■ありがとうのプレゼント
も「次の曲はセルフカバーです」
も「わたしが 4 年前にレコーディングしたソロ曲、ありがとうのプレゼント」
→ てんかす 3 人で披露
みんな歌がうまいけれど、やっぱりこの曲は有安さんだなーと思いました
有安さんのソロパートも、4 年前の CD 音源よりもだいぶ成長した感じがしてとてもよかった

■発表
「ももかちゃん、発表することがあるんだよね？」
も「ちょっとその前に水飲むね（・Θ・）」
→ ステージの奥のほうにスタスタと歩いていき、ドリンクをチュー
も「てんかすトリオで前からお願いをしていた、新曲をレコーディングしました！」
も「今日はその曲を宇宙一早くみなさんに聴いてもらいます」
→ 新曲 Puzzle きたー！
→ モニタに作詞が谷山浩子さんとか出ていたようですが、自分の席からは見えませんでした…

■Puzzle
曲調はトリニティー同様、アニメのテーマソングになりそうな感じ
ダンスがかなりキッチリと振りつけられていて、こちらもトリニティー同様に凄かった

■永遠のトリニティー
も「さっき、ねごかすでやったけれどダンスが無かったからもう一回」
→ もう一回きたー！
藤井ベストテンで 3 人そろったパフォーマンスが出来なかった分、今日きちんとやってくれて嬉しかったです
自分は去年のグリーンドーム前橋の藤井の映像を何度も見ているのでそのイメージが強いんだけど、それよりもさらにパフォーマンスが研ぎ澄まされていたと思います

■フィナーレ
全員で一列になって合唱
→ 有安さんは上手側はじっこのほうに行ってカメラにアピール
→ かと思えば、下手側一番はじっこでもアリーナ＋スタンドにレス＋カメラにアピール
…といった具合に、結構位置が変わっていたので大勢の中からずっと有安さんを追っかけてみてました
→ なんとなく、お遊戯会であっちこっちに移動する自分の子供を追っかけて見ている保護者の気分ｗ

22:00 ちょうどに終演

■まとめ
4 時間立ちっぱなしでしたが、予想以上に有安さんの出番が多く楽しかったです。
他のグループも全部ちゃんと観ましたが、ももクロの 5 人は性格・声質などのバランスが抜群だなーと思いました。
そして、自分はやっぱりももクロの 5 人が好きなんだということを再認識。
中でも、有安さんが好きだからももクロが好きなんだなーと改めて感じました。
今日も有安さんが進化していることを実感できたし、自分はよそのグループに推し変することはなさそうです。

■写真
終演後は撮影可能でした
