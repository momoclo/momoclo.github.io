転校生 アフタートークショー

転校生アフタートークショーに行ってきたよ。
避難所メモを元に、トークショー関連だけ思い出せる範囲でレポート。

■会場
Zepp ブルーシアター六本木
舞台「幕が上がる」をやった場所
前方ブロックは つぶしてメインステージ＋機材席に
客席は後方ブロックのみ

■0場
18:30 スタート
伊藤沙莉さんは「幕が上がる」と書かれたTシャツ姿
他にも「KSDD」と書かれたシャツのひともいました

18:45 過ぎに ももクロちゃんも全員登場
衣装は富士ヶ丘高校の制服

設定は久しぶりに学校に来た 5 人組が友達と再会するといった感じ？
か「この制服、久しぶりに着たら若干キツイ…」
→ ちょっと太ったとか言ってました

れにちゃんは相変わらず がるる風
しゃべった順だと、れ ＞ か ＞ あ ＞ し ＞ も という具合
→ ももかちゃんは中西さんっぽい感じでほとんどしゃべらず

がるると高田さんで行くぜっ！怪盗少女のサビを振り付けで歌うというサービス？があった

■れにちゃん
基本制服姿だけど、れにちゃんだけ がるる風に下半身ジャージ
左手はサポーターをしてなかった

■タイムテーブル
19:00 本編開始
20:25 本編終了
20:30 トークショー開始
20:57 トークショー終了

■音楽
ステージセットのピアノを演者が生演奏
幕が上がるで使われた曲もあった
→ 銀河鉄道の夜のシーンで流れた曲かな？
→ しおバンニが ももカンパネルラに「クルミ、ぼくもずっと持っているからね！」のシーンあたりの曲だった気がする
→ 曲名を確認しようと、サウンドトラックをチェックしたけど入ってないんだっけ…

■同時多発会話
本編の 80 分のうち、70 分くらいは同時多発会話
今日が初見ですが、ストーリーはさっぱりわからなかったです
終盤、いきなりクライマックスが来てそのまま終わる感じ

トークショーでれにちゃんが「オリザさんの同時多発会話というワザ？で話が難しかった」と感想を言ってました
→ 「ワザじゃなくて手法」とあーりんに言われていたかな？

本編の最後は BGM 青春賦でした

■トークショー
ももクロの茶色担当、ちゃいりこと伊藤沙莉さんがひとりで登場
沙「どうも、司会進行をつとめます伊藤沙莉です」
沙「えー、それでは さっそくお呼びします！」
→ ももクロちゃん登場
ももクロちゃんに呼ばれて本広監督も登場

■パラレルワールド
映画・舞台・転校生はパラレルワールドの設定
沙「構想当初は『サワダ』で、映画当初は『高田圭子』で、映画と舞台は『高田梨奈』で、転校生は『高田美幸』で、名前が 4 通り違う」
本「パラレルワールドってやつですよ、わかる？」
→ かなこちゃんに視線が集中すると…
か「わかった！」
か「ドラえもんみたいな世界！」
→ 確かに世界線がずれていれば、ドラえもんがいる世界があってもおかしくないよなーと思っていたけれど外れ

本「なんでパラレルワールド設定なのか、わかる？」
→ かなこちゃんに視線が集中すると…
か「わかった！」
か「あーりんが二年生で明美ちゃんがいるはずなのに、転校生にはいない！」

なかなか鋭いなーと思いましたが外れらしく、本広監督から正解が
本「さおり部長が髪の毛を切っちゃったので、そこからパラレルワールドにすることになった」
時期がずれていれば髪の毛の長さが違っていてもいいような気もしますが、PV ではポニーテールのカツラをかぶったり、色々と調整していたようです

沙「パラレルワールドのせいで、あたしは名前がコロコロ変わることに…」
→ ということで、かなこちゃんが謝罪することに
か「みんなも一緒に謝ろう」
か「監督も一緒に謝ろう」
→ みんなで大きな声で謝ることに
一同「どうもすみませんでした！」
→ かなこちゃん、お辞儀の角度が前屈体操並みに深かったです
「前屈苦手なのにちゃんと謝ったねー」

■おっぱい
舞台を見た感想を訊かれたももクロちゃん、「誰が印象的だった？」の問いに
か「おっぱいのひと！」
→ おっぱいのひとが呼ばれるｗ
→ キャストを確認したら、川面千晶さんかな？ (検索した結果と印象が違うけど、合ってる？)
→ この時、有安さんが「(かなこちゃんが？) さっき裏で触ってたよね」とつぶやいてましたｗ
おっぱいのひとは映画で吉岡先生の後輩役 (アゴラ劇場を見学に行ったときに登場) の方でした

■有安さん
ももかちゃんは ももクロちゃん＋ちゃいりさん＋監督のトークバトルを笑顔を聞いていることが多かったんだけど…
か「さて、ここで声を聞いていないひとにも感想を聞いてみようかな」
→ と、有安さんをターゲット
ももかちゃんは突然ふられて、セットの教室机に突っ伏してましたｗ

ここまでほとんどしゃべっていないことを指摘されると…
「そもそも、マイクの位置がみんなは口もとにマイクを持っていってて、『うんうん』とか言ってる」
「けど、有安はマイクの位置が下になってる」
→ マイクを持った両手を体の前に下ろしている状態だったので、そういえばマイクに声が入らないですねー

も「ちゃんと話は聞いてました！」
も「やる気はありまぁす！（｀・Θ・´）」

■青春賦
有安さんはこうやって話をふられる前にも少しはしゃべってました
も「本編の最後に青春賦が流れるけど、あれは毎回そうなの？」
沙「本番の初回まで知らなかったけれど、毎回そう」
も「今回だけサービスかと思った」
→ 稽古の時は流れていなかったらしいですが、本番では毎回やっている模様
→ 演者のみなさんにとっては この曲が「弱点」らしく、エコパスタジアムで泣いたエピソードが話題に出てました

■青森中央高校
ももかちゃん、印象に残った件を訊かれて…
も「映画で一緒にまくら投げをした、早紀ちゃんが出てる」
→ 折舘早紀さんがステージに登場
→ 「この性格ブス！」とか言い合っていた子かな？
早「映画の静岡ロケの時、青森に帰る時には『ももかちゃんやさしい』と思って別れるのがつらかった」
早「あれから、青森中央高校にはモノノフがいっぱい増えた」
早「青森中央高校の演劇部では、稽古前のウォーミングアップで Chai Maxx をみんなで踊っている」
→ 早紀さん自身はもう卒業しちゃったので、そういうのがうらやましいというニュアンスだったかな？

■フィナーレ
20:50 すぎに本広監督が裏に引っ込んだと思ったら、出演者全員を連れてステージ上へ
最後はステージに全員が整列し、あらためて「ありがとうございました！」
→ ももクロちゃんたちはゲストということで、センターを出演者に譲って端っこにいました
最後に司会進行ちゃいりさんが締めるように促されて…
沙「えー、宴もたけなわではございますが…」
沙「BGM が流れてきて、早く終われという空気なので、このへんで終わりにしたいと思います」
→ BGM は青春賦でした
みんな笑顔で手をふりつつ退場

最後の最後、裏手に消えそうになっていた れにちゃんとちゃいりさんが連れだってステージ中央へ来てお辞儀して去って行きました
→ がるる子弟コンビですねー
