六本木LVに行ってきたよ。

■観客
いつもの六本木LVとさほど変わらず。
今日は白秋のように落ち着いたアコースティックライブだったのでコールはバレンタインイベントよりもさらに控えめ。

■開演
ゴールドベルグ変奏曲っぽいピアノ曲が流れる大人な雰囲気
http://youtu.be/1PBlY3USDXA
19:00ちょうどに開演
GOUNNツアーのナレーションを担当した久米明さんの「大きな古時計」の歌詞(1番かな)朗読からスタート

■衣装
5色のスカートに白いトップスのドレス
それぞれのソロパートではコスプレ
アンコールは日産スタジアムの時の5色ドレス

■音響
マイクはワイヤレスではなく、ワイヤード
音量バランスなんかも「歌」を聴かせることを重視していたように思う
正直、一番声が出ていて歌がうまかったのはゲストの辛島さんだと思うけど、その次は間違いなくももかちゃん
→ 辛島さんといい勝負と言ってもこのスレならアリかな

■セットリスト
ももクロちゃんの曲、メンバーソロ曲をアコースティックアレンジ
その他カバーも色々
しおりん → 尾崎豊さん「卒業」白いTシャツにジーンズ、髪もアップにしてオールバック風
あーりん → 松田聖子さん風の茶髪ウィッグで80年代アイドル風、足はまだギプスをしていたけどダンスも少しずつ復活
れにちゃん＋かなこちゃん＋しおりん → アルフィー桜井れに、坂崎かなこ、高見沢しおり編成
みんなコスプレがよく似合ってた
れにちゃんはアルフィーと共演した時と同じくサングラス＋口ひげマジック
ベース、ギターともに弦がびろんびろん どう見ても弾いてないだろ！という有様だけど楽器も持って本格的

■ももかちゃん
普通のストレートヘア＋前髪おろし
前髪はだいぶ伸びてきてて、目にかかりそう

■永遠のトリニティー
ソロ曲を披露する流れで、ありがとうのプレゼントが来るかなーと思っていたら「永遠のトリニティー」だった
自分は俺の藤井の現場で一度見たのでわかった
衣装は俺の藤井と同じ青＋黒のゴシック風ドレス
一部、物理的にひとりじゃ歌えない箇所をコーラスのマロンさんがフォローする形ではあったけど、一人で歌いきった
→ メンバーから「3人じゃなくてもよく歌えるね」と言われてた

■ももかつぜつ
(・Θ・)「しゅつえん」
→ ん、今なんて言った？

→ んんん、今なんて言った？？？
→ れにちゃん「キジちゃん (じゅるじゅるさせながらモノマネ)」
→ れにちゃんが何度もモノマネして遊んでたｗ

■ゲスト
辛島美登里さん「愛と恋の違いについて」
→ 各メンバーに違いはどこか質問
→ ももかちゃん「恋は…ドキドキ、愛は…なんていうか産まれる！？ ん〜言葉にするの難しい」
各メンバーそれぞれ違った表現をしていたけど、割と言いたいことは共通している感じ
恋は1対1で、時にはギクシャクする
愛はみんなで、まわりを暖かく包み込む

■アンコール
みんなで歌おう！という呼びかけからのオレンジノート
アコースティック＋ももクロちゃんマイクなし生歌
それに加えて観客が一緒に歌う
→ LV会場でもみんな歌ってた
最後のサビは演奏一切なしでアカペラ＋観客コーラス

最後のZポーズで終わりと思いきや、さらに灰とダイヤモンド
→ 終わってそのまま ももクロちゃんがバックステージにはける
→ れにちゃんから上手に消えていったけど、あーりんやすがステージに残ったときにあーりんがLVカメラを指さして…
→ あーりんやすでLVカメラに手を振ってくれながらの退場

■終演
久米さんの「大きな古時計」歌詞後半(2番かな)朗読
「今はもう動かない その時計」で暗転して21:50終了
なんだかんだで3時間近いステージでした

■終演後
六本木LVでは何故かなかなか場内の照明が明るくならず
この後まだLV挨拶でもあるのかなー？
それとも、ままままさかのももクロちゃん登場が！？
と思って帰らないひと多数
結局10分くらい謎の期待感で待っていたけど22:00に劇場係員から「もうこれで終わりです」とのアナウンスがあり終了

■感想
「歌」に真剣に取り組むももクロちゃんの姿と、表現の幅を広げようという意気込みが感じられるステージでした
まだまだ先輩歌手には劣る部分もあるけれど、がんばる姿は変わらずチャレンジしているなあと改めて実感
→ かなこちゃんもMCで少し「歌はまだまだだけど、がんばります！」的なことを言ってた

永遠のトリニティーは俺の藤井では現場で遠目でしか見られなかったけど、今日はLVで表情までよく見れてとてもよかったです。
今更だけど、やっぱ歌うまいわー
