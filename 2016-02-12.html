川上アキラのオールナイトニッポンR

川上アキラのオールナイトニッポンR 公開収録に行ってきたよ。
NDA ぽいところは伏せつつレポートします。

■会場
科学技術館サイエンスホール
日本武道館の近く
キャパは 400 人くらいかな？
縦長の映画館といった大きさ
椅子は飛行機みたいに、前の席の背中部分に折りたたまれているテーブルつき

■観客
男女比 → いつものライブよりもだいぶ男性が多い
年齢層 → いつものライブよりも若干年齢が高いような…
服装 → 会社帰りのひとが多く、みなさんかなり地味
ということで、単なるおっさんの集団みたいになってましたｗ

■前説
18:50 スタッフが登場して拍手のタイミング説明と、ジングル〜タイトルコールの練習

■開演
19:00 開演
トーク内容は放送前なので自粛

放送内容とあまり関係ないところだと…
曲を流す間の時間はテレビの CM 中トークみたいな感じで、川上さんが裏トークをしてました
川「こんな感じでいいんですかね」
川「こんな話でいいんですかね」
→ それほど内容があるわけではなく、節目節目に「大丈夫？」と観客に確認する感じ

曲自体は最初のワンコーラスの途中で終わっちゃったので、新曲をばっちり聴ける！というわけではなかったです
ピンクゾラだけ少し長めにワンコーラス全部くらい流してました

21:03 終演

■ももクロちゃんホットライン
終演後、川上さんがスタッフに確認
川「21:10 くらいまでいいですかね？」
ス「オッケーです」
→ 川上さん、裏にひっこんで携帯電話を取りに戻ってました

ステージに再登場後、電話をかけようとするものの…
川「あれ、圏外だなー」
川「ドコモなんだけど、みんなのは使える？」
→ 観客席ではオッケーですよー！
→ 川上さんが観客エリアに降りてきつつスタッフに電話

無事に電話がつながり、スタッフが電話口に
川「誰でもいいんで、誰かいる？」

■しおりん
川「みなさんの大好きな川上ですが、ミュージックステーションどうでした？」
し「そんなに好きじゃない」
→ ばっさりｗ

■あーりん
あ「みんな、川上さんを見るためにお金を払ってきているってこと、わかってんの？！」
川「これからは ももクロさんに偉そうなことを言わないようにします」

■れにちゃん
何かテンション高めに色々と話してくれましたが、声質のせいなのか、機材の調子のせいなのか、エコーが輻輳しているような感じで聞き取りにくい状態
川「…有安さんのマネですか？」
れにちゃんか、あーりんか声が似ていてわかりにくかったですが、「(電話を)切らないで〜(はあと)」というコント？をやってました

■ももかちゃん
川「チビ太さんですか？」
も「そうだよー」
も「お客さんの声が聞こえるー（^Θ^）」← 声の調子が嬉しそう
川「ミュージックステーション、どうだった？」
も「めっちゃ緊張した！」
→ その後なにかムニャムニャ言ってましたが、よくわからかったですｗ
→ きっと機材の調子が悪かったんですね

川「それじゃちょっと緑推しに代わります」
→ 近くにいた緑推しの女性にバトンタッチ
緑「ソロコン待ってる！」
も「待っててねー！」
→ いえーい！

■かなこちゃん
川「最後に何かいいコメントを」
か「バレンタインデー キッス〜♪」
→ 唐突に歌い始めるフリーダム百田さんｗ
か「チャララ ラララーラー」
か「わっしょい！」
→ 歌の最後をお祭り風に締めるリバティー百田さんｗ

■ミュージックステーション
ももクロちゃん「そこのお客さん、ミュージックステーション見てないの？」
川「見てないんじゃなくて、Mステ の発表前にこっちのチケットを取っちゃったから見られない、かわいそうなひとたちなんだよ」

■ジングル
最後に電話がつながった状態で、またまたジングル〜タイトルコール
川「川上アキラの〜」
→ オールナイトニッポンあーる！
→ ぱちぱちぱち (拍手)
→ さっき、本番用にがんばって録ったはずのものよりもみんなの声＆拍手が大きかったですｗ

■お約束
このへんで 21:10 過ぎになり…
電話の向こうではももクロちゃんが何かしゃべってましたが、川上さんが電話をぶつ切りｗ
川「あ、切っちゃった」
川「最後にいつものあれをやっておしまいにしましょうかね」

川「以上、わたしたち ももいろクローバー」
→ ぜーっと！
→ メンバーが誰もいないところで勝手にやって締めて終了ｗ 21:15

■まとめ
先月の俺の藤井から丸一ヶ月ほど海外に滞在していたので、最近のトレンドは追っかけきれておらず「切らないで〜」ネタあたりは状況がよくわからずに書いてます。
電話は各メンバー同じくらい色々しゃべっていた気がしますが、どうしても有安さんの話を集中して覚えようと思ってヒアリング＆メモしているのでレポートはちょっと多め。
だいぶ前にソロコンサートの話が出たときは「まだ準備が全然なので、川上さんが企画しないように全力で止めている」とか「準備に三年かかる」と言ってましたが、やっぱり前向きに考えていてくれているのかなー？
楽しみに待っています！
