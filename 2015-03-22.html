女川町復幸祭

女川町復幸祭2015に行ってきたよ。

■交通
仙台駅からクルマに便乗させてもらいました どうもありがとう
6:45 仙台発 途中コンビニ休憩を挟んで 8:20 女川の高政着

帰りもクルマで仙台まで 思ったほど渋滞はひどくなく、3 時間くらいで到着

■高政
仙台駅からクルマで行ったので、途中で立ち寄り
自分は去年の夏に続いて二回目
開店時間の 8:30 少し前に着くと、観光バスの団体客がちょうど来ており、少し早めに入店できた
ももクロちゃんたちが座ったテーブルで、同じく焼き笹かまを注文

■出店
駅前広場には様々な出店が出ており、昨日の数倍の人数でにぎわっていた
「笹かまでお友達」をゲット
その他、いろいろと食べてきました
ゆるキャラが総勢 10 人くらい来ていて、たぶん全員とツーショットを撮影しました

■会場・天気
サッカー場のような多目的運動グラウンドで人工芝
ステージの高さはやや低め 最前列が小中学生エリアだったからかな？
雰囲気としてはクノっぽい感じ
音響はよくなかったけれど、立地や手間、最前がチビッ子エリアということ等で文句はないです

駐車場のクルマのナンバーを見ると、東日本のあちこちから来ていた
→ 見た中で一番遠いのは愛知ナンバーだった

先週末の週間天気予報では雨模様だった予報が、日が近づくにつれてどんどんと降水確率が減り、最終的には晴れになってた
→ 帰りの新幹線では仙台を出たところで少し雨が降ってました

■開場前
山ステージに 11:00 過ぎに到着すると、列形成がはじまったところ
「整理番号順に並んでくださーい」という指示はあったものの、どこがどのブロックで、どこらへんが何番なのかさっぱりわからずｗ
それでも、まわりのひとと確認しあったりしてビックリするくらいスムースに入場できた

並び始めはこのカオス状態だと、トラブルが起きたらやだなあと思っていたけれど、まったくトラブルは見かけなかった
あの人数と不慣れなスタッフさんの指示で混乱なく入場できたのはミラクルだと思います

11:30 から入場を開始して、Bブロック 1400 番台の自分たちが入場したのは 12:00 過ぎ
12:30 には完全に入場が終わっていた感じ

BGM は三味線＋若干ロックアレンジの吉田兄弟っぽい曲調
天気はとてもよく晴れており、雲ひとつない青空
それなりに寒さ対策をしていった格好だと少し暑いくらいのポカポカ陽気
上空には取材？のヘリコプターも飛んでいた

■開幕
13:00 Overture でスタート
かなこちゃん、開口一番に「女川町のみんなー、ももクロが遊びにきたよーっ！」と高らかに宣言
→ だいぶ前に「女川のみんなとは友達なので、ふらっと遊びに来るだけ」と言っていたよね

■衣装・髪型
仮想ディストピアの印象が強い、横ボーダーのセーラー服っぽいアレ
ももかちゃんはツインテールだったと思う
ステージがほとんど見えなかったので、表情・しぐさ等についてはレポート少なめ

■自己紹介
も「有安杏果、ハタチです！」
→ 時間がタイトなせいか、割とさらっと
→ ノフからは「おめでとう！」の声が挙がってた

MC の順番は覚えていないので、覚えている小ネタをレポート

「コフレドールっぽかった」

■セットリスト
元気が出るような、前向きな曲ばかりでとてもよかった
女川町の方にもぜひ歌詞を確認しながら聴いてもらいたいところだけど、音響とかコールとかで伝わったかなー？

■サラバ、愛しき悲しみたちよ
…の前の MC で、女川にまた来たいねーといった内容のトークをしており、時間的には中盤だったものの話がなんだか締めっぽい感じに
あ「次の曲がサラバなんで、この流れでサラバだとなんだかおかしくて笑っちゃった」
→ セットリストがバレてますｗ

「あ、次の曲についてはみんな知らないふりで、始まったら『おー』という感じで！」
サラバのイントロがかかり…
→ おおおおおおー！

■ニッポン万歳！
有安さんの「宮城の牛タン」は去年の めざましライブ同様、気合がこもってた
12:50 本編終了

■アンコール
すぐにアンコール
…と思ったら、いつの間にか れにちゃんだけステージにいた
→ 他のメンバーが準備？をしているあいだ、「適当につないでおいて」という役だった模様ｗ
れ「一曲うたいます！」
れ「特別な おーながーわ〜 わーたしにくーれる〜 あったかいんだからぁ〜♪」
れ「わたしたちのことも、町民として認めてくれますかー？」
→ いえーい！

などなど、適当ｗにつないでから…
れ「それではこのへんで、特別ゲスト、ももいろクローバーZさんです！」
→ れにちゃんがももクロを呼び込むという展開ｗ
あ「前代未聞だよ！」
か「もう帰るつもりで、歌う気がまったくなかったんですが…」
か「みなさんのアンコール リクエストの熱いパッションにお応えして、あと二曲うたいます！」

■ツヨクツヨク
最後のタオルの向きは見えませんでした どうだったんだろ？

■青春賦
女川小学校・中学校の生徒さんがコーラスで参加
さらに、女川さいがい FM のアベさんが引率の先生として登場

登場時・イントロ・アウトロ・退場時すべてで拍手が起こってました

ア「今日は ももクロちゃんたちに、感謝の気持ちを込めて一生懸命練習してきました」
「初コラボだね！」
あ「小さなお友達も、大きなお友達も、一緒に歌ってね！」
→ はーい！

チビッ子の中には女川さいがい FM のリスナーさんから送られたピンクの衣装を着ている子がおり (よく見えなかった)、その衣装の出来栄えが素晴らしかったようで…
あ「その衣装、誰かに着せられてない？」
→ 誰かの圧力でピンクにさせられているんじゃないかと心配する圧ーりんｗ
あ「自分の本物の衣装、倉庫にあるかな？」
→ 持って行かれちゃったんじゃないかと心配？するほどだった模様

れ「そろそろ世代交代だよ！」
あ「高城さんには言われたくないわ！」

こんなやりとりをしていた横で…

■帰り道で緑推しさんから聞いた話
あーりんの衣装の話をしていたときに、ももかちゃんは自分のそばにいた小さな子のところでしゃがんで、なにか話しかけていたそうです
→ とっても緊張していたチビッ子に「大丈夫だよ〜」とか言っていたっぽい

その後、他のメンバーから「有安さんのほうはどうなの？」と振られて
も「こっちもすごく緊張しているので、みんな大きな声で歌ってね」

■友よ
有安さんのセリフパート、「友よ この切なさが」の部分が CD とはちょっと違うニュアンスだった気がする 語り掛ける感じ？

■退場
「また来るねー！」
最後に れにちゃんが「女川さいこう！」

■まとめ
駐車場もあちこちに充分な数を用意していただき、シャトルバスも様々なルートで頻繁に運行してくださったのでとても便利でした。
女川町にはまだまだ瓦礫の山がありましたが、ももクロちゃんとモノノフが非日常的なカラフルさで殺風景な中に少しでもアクセントを与え、地元の方々にイベントで来てもらってよかったなと思ってもらえたなら嬉しいです。

関係ないけれど、ももクロちゃんに連れられて女川町まで、というのをハーメルンの笛吹っぽいなと思って改めて調べたら、カラフルなところが同じでしたｗ

ももクロちゃんたちは、あの 5 人じゃないとできないやり方で女川町を応援しているんだと思います。
これからも、あの 5 人が笛を吹き続ける限り、ついていこう、そう思いました。
