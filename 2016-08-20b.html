ロケハン青森 2 部

ロケハン vol.7 青森 2 部に行ってきたよ。

■会場
1 部と同じオルテンシア

■タイムテーブル
16:00 開演
17:35 終演

■0場
開演前にれにちゃんが舞台袖からひょっこり顔を出す…
その後、上手から下手までダッシュで走り抜ける
→ 「おのれ、グッチー！」と言っててもおかしくない展開でしたｗ

■M
通常版 Overture で開幕して、れにちゃんがステージ中央へ
れ「それではきいてください、プリンセス・プリンス」
→ プリンセス・プリンセスですねｗ
れ「M」
→ すぐに歌い出したら、あとからイントロが…
→ 今日の音出し担当は古屋さんではなく、カニパンさんだったらしい
→ 1 部でも音出しのタイミングをミスったり、曲を間違えたりと散々でした
れにちゃんは歌詞がよくわからなかったらしく、有安さん・あーりんにヘルプを求める
→ 有安さんが登場して一緒に歌唱
あ「まだまだいけるよー」
→ あーりんは舞台袖でずーっと見てました
あ「あれに出て行ったら事故ると思って、絶対出るもんかと思ってた」

■桃神祭の裏話
ボックスから「桃神祭」のお題
あ「楽屋で流しそうめんとか、かき氷とか夏らしいのをやった」
も「ゴリラパンチの歌詞まちがえた…」
→ 「クラゲみたいに」を「サボテンみたいに」にしちゃった件ですねー
も「桃神祭が終わってから、ありやすサーチをしたら、みんなの感想がそればっかりだった！」
その後、歌詞を確認してからゴリラパンチをやり直し披露

川「有安は氷をバリバリ食べてそう」
も「食べたことないもん」

■ジャンボマックス
桃神祭で出てきた、大きな有安ロボ
もともとはドリフのネタらしい？ (世代ではないので、よくわからず)
なぜ有安さんだったかというと、ブー雷様が緑 → 緑ならウチの有安で、ということらしい
川「あれ作るのにお金かかっちゃったので、どこかで出そう」
も「どうせ子供祭りだろ！」

■Tシャツ
も「桃神祭の T シャツ、プリントがハゲちゃってごめんねー」
も「ハゲちゃって」
→ ハゲ、ハゲと言っていたので、それはよくないのでは…？と川上さんから指摘
も「ぷ、プリントが薄くなっちゃってごめんねー」
→ フォローになっていませんが…ｗ
も「やばい、自分で墓穴掘った」
→ その後、最終的には「プリントが落ちちゃってごめんねー」に落ち着いてました

■桃神祭の懺悔部屋
ここから桃神祭でミスをしちゃった告白コーナーに
あーりんは サラバ、愛しき悲しみたちよ で歌詞を飛ばしちゃったとのこと
れにちゃんはゴールデンヒストリーの冒頭のダンスを忘れて棒立ちｗ
→ その他にも、観客から しおりんのミスなどの指摘
「細かいところまでバレてる…」

■寄せ書き
1 部の開演前に地元？ノフさんが募集していた寄せ書きがステージに
→ 小さなハート型の紙に書いたメッセージを大きな模造紙に貼って展示
→ 「ようこそあおもりへ」
→ もあかしれ の似顔絵風イラスト

ももクロちゃん、開演してすぐにそれを見入ってましたが…
→ みんなボードのほうを向いてじっと読んでいる図ｗ
川「タレントさんがコメントを一切読み上げないですね」

■GTO
Great Takahashi October
れにちゃんが発案
ももクロちゃんたちは Gig Takahashi の話題と勘違いしており、川上さん・観客ともなんだか話が噛み合わないｗ

■ロケハン
有安さんがリクエストボックスからお知らせを引き当てる
vol.9 北海道 9/25
vol.10 三重 10/1
で合ってるかな？

最後にカニパンさんがれにちゃんに何か耳打ち
→ 来週の鳥取は 20 分時間が早まるそうです

■衣装・髪型
かき氷をイメージした桃神祭衣装
→ 帽子のサイズは自分で選んだ
→ れにちゃんは小さめ、あーりん・有安さんは大きめの麦わら帽子
有安さんはツインテールを三つ編みにして、さらに三つ編み部分で輪っかをつくった複雑な髪型

■4年後
桃神祭はオリンピック周期、というあたりから 2020 年の話
れ「その時には 27 歳だ」
れ「どうなってるか想像つかない」
も「それぞれ、27、25、24 歳になってるね」
あ「まだ若いな」

■自己紹介
会場から赤、黄色担当を召喚
→ しおりん担当は小学生の男女
→ かなこちゃん担当は大人の男女

■フリコピ
仮想ディストピアを ももクロちゃん＋会場選抜 5 人くらいで

■腹黒い
川「(れにちゃんのことを) このお姉さんは、こう見えて腹黒いからね」
→ ちびっ子に謎の解説
れ「バラさないで」

■のへ丼
1 部が終わったあと、近くのマルコーセンターで念願の のへ丼を食べてきたそうです
あ「今日はもう 3 食目だった」
あ「川上さん、今日の 3 食とも見てるよね」
→ 朝の飛行機で 1 食目、会場について 2 食目、のへ丼で 3 食目だったっけ？
川上さんはそれぞれ 2 人前ずつ食べてたりする模様

■モノノフロス
れ「桃神祭が終わって一週間、モノノフロスでしたが、今日のイベントでだいぶ補充できました！」
→ なんだか立場が逆な気がしますが、そう言ってもらえると嬉しいですね

■テッパンです
最後にぜーっと！で全員がはけるときに…
あ「青森リンゴはー？」
→ テッパンです！
→ 有安さんじゃなくて、あーりんが拾ってくれた気がします
→ 有安さんも気にしてたようだったけれど、最後にあーりんがねじ込んでくれたのかな？

■おまけ
日曜日に早起きして、のっけ丼センターをパトロールしてみましたが、有安さんは来なかった模様

AUGA 地下の郷土料理 津軽三味線 りんご箱のサインと、スタッフさん・推され隊の 3 ショットは発見しました
→ のっけ丼が食べられなかったので、代わりにのっけ丼モドキを作ってくれたお店
→ サインと写真はお店の営業時間前に訪問して撮影

昼食は青森県観光物産館アスパム 10F の西むらで、つがる定食をいただいてきました
→ れにちゃんがプライベートで来た、といってお座敷から外を眺めていたお店
→ れにちゃんが座った席もわかったけれど、他のお客さんがいたので写真は撮れず
→ サインは撮影させてもらいました
