国立競技場に行ってきたよ。

シングルメドレーまでは昨日と同じ、最後の挨拶以外ではMCや茶番もなしの、終わってみればガッツリライブでした。

■ボール投げ
昨日と同じ調子ｗ
スタンドに届くとおおー！と歓声が(あがっていた気がするけど、自分だけかもしれない)

■ももクロのニッポン万歳！
リヤカー型トロッコで5人で一周
ボールは前半で投げまくって、後半は弾切れ気味だったかな？
トロッコはランウェイとの段差が20-30センチくらい、スピードもかなり遅かったのでちょいちょいトロッコから降りてランウェイでレスを配ったりしてた
→ 何故かあーりんに追いかけられる
→ その後立場逆転であーりんを追いかけてたけど、足の速さはいつも通り
→ ランウェイからトロッコに戻って乗ろうとしたら、(たぶん)かなこちゃんにブロックされるｗ

■19歳
かなこちゃん「なんだか、19歳になった有安さんはカッコいいね！」
→ 有安さん調子に乗ってLVカメラにアピール
今までとちょっと違う一面を見た気がした

■聖火台へ
全曲終了後、メインステージには5人の姿のみ
→ モニタには れにちゃんのソロ映像が流れ始める
→ そしてゆっくりと、ひとりでサブステージに向かって歩いていくれにちゃん
→ 会場は紫一色
→ 割と唐突だったので、少し不吉な予感がした
…けれど、その後ひとりずつ同じ感じでサブステージへ
→ あせったけど安心した
ももかちゃんの時はグリーン一色の絨毯を歩いているよう
→ 既にでているけれど、風の谷のナウシカぽかった

サブステージに5人そろうと、開幕でダイブしてきたジェットコースターに搭乗
→ そして聖火台まで上昇

■最後の挨拶
5人とも、とても素晴らしいコメントをしていたのでどうせ書くなら5人分を、とも思ったけどスレッド違いなので自粛
かといって、有安さんのコメントをレポートするのは自分自身胸がいっぱい＆中途半端な記憶で書くと正確に伝わりそうにないのでうまく書けない…と思ったらナタリーさんが。
http://natalie.mu/music/news/112182

書き起こしで伝わりにくいニュアンスとしては、本当にやさしく語り掛けるような感じだった
→ 「みなさん、今日は楽しかったですか？」とか

■有安さん
昨日、今日と、ももかちゃんの嬉しい楽しい幸せという気持ちが応援しているこっちにも伝わってくるように感じた
これって、まさしく ももパワーそのものだよね…

■夢の向こう
昨日はこれまでの集大成、今日はこれからの未来というコンセプトがしっかりわかる構成だった
ゲストとしてタカミーが来たけれど、ゲストの力を借りることなく、昨日よりも堂々としていて、5人でしっかりと国立競技場を自分たちのステージにしていたと思う
会場の大きさとか、新たな壁とか、そういうことじゃなくてこれからもずっと一緒に前に進んでいければいいなと思うと同時に、全速力で駆け抜けるももクロちゃんについていくなら、こっちも振り落とされないようにしていこう、と感じさせる逞しさがあった

■日産2days
7/26(土), 7/27(日)はカレンダーに日産スタジアムライブ 絶好のライブ日和で大成功！と書いておくことにするよ！
