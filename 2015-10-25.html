家康楽市

第8回家康楽市in浜松出世城「2015秋の陣」に行ってきたよ。

■序
お台場ハロウィーンから幕張メッセの Halloween Party に移動中に川崎の黄色推しさんと阿佐ヶ谷の赤推しさんに誘っていただく
行くつもりは全然なかったけれど、クルマに同乗させてくれるということで急遽行くことに
日曜日の朝 6:30 鷺沼駅集合

■第一部
9:35 頃、会場の浜松城公園に到着…すると遠くからでもわかる、聞き覚えのある声が風に乗って聞こえてきた
→ 他のひとの声はほとんど聞こえなかったけれど、周波数が高いせいか？風によく乗る＆遠くからでも判別できた
→ あわててステージまで小走りで移動

観客は半分くらいがモノノフ？
→ 当然のことながら赤推し率が高く、他の色は数えるほど
→ 自分も赤いTシャツ＋赤いウィンドブレーカーで行きました (元々原色好きなので赤・青・黄色・緑あたりの服はたくさん持ってる)

お天気は雲ひとつない青空 かなこ晴れですね
→ 風が強かったけど、遠州からっ風というらしい
→ 気温が低めの割に日差しがあったので、日中は寒くなかった T シャツで充分
→ 日が落ちてからは急に寒くなり、上着を着ました

かなこちゃんは赤い甲冑姿
→ 聞いたところによると、重さ 20kg とか
→ 本物だとしたら、自分の体重の半分近い重さの衣装を着ていることになりますねー

か「ゆるキャラの家康くんとは何度か共演しているけど、しゃべってるバージョンは初めて見た」

随分と朝早くから出演するんだなーと思ったら、10:30 予定が繰り上がったらしい
→ 1 時間繰り上げで、あの衣装で出演ということは相当前から会場入りして準備しているんでしょうね
→ 8 時くらいに会場入りしていたという話

トシ子ちゃんも交えたトークをしつつ、チビッコ向けに餅撒きをして 9:50 退場
→ Twitter TL などを見ると 9:30 前くらいから出演していたらしい？

■第二部
12:00 チビッコのパレードにまじって登場
なんと黒い馬に騎乗しての登場
今度は狩りに行きそうな感じの白い袴っぽい衣装＋烏帽子

馬に跨った状態でしばらく撮影タイム
→ 撮影が終わると、馬から降りて歩いて退場 12:10

■幕間
一番大きなステージでは、かなこちゃんが出た催し物以外にも和太鼓の演奏や、ご当地アイドル？みたいな戦隊モノっぽい武将隊が歌を歌ったり
モノノフはいつ かなこちゃんが出てくるかわからないので、メインステージを張り込む形にｗ
→ おかげでどの演目もかなり注目度が高かったと思います

武将隊は「グレート家康公 葵武将隊」というらしい
→ 戦隊モノのテーマソング系・バラード・ラップなど多彩なジャンルを歌って踊ってましたｗ
構成メンバーは家康公とその家臣の合計 4 人
→ うちひとりは赤い甲冑で、「レッド」が担当カラーの模様
レ「本日は大勢のお客さんが来てくださり、感謝でござる」
レ「中でも、赤い格好の方が大変多く、赤担当としては嬉しい限り」
レ「今日はこの後、赤界のトップに君臨するあの方も出演なさるので、ぜひお見逃しなく」

他にも、なんだか関係のありそうなワードがちょいちょい出てました
→ 「傾奇者」
→ 「(振りコピみたいなのを一緒に踊る曲の説明で) 見よう見まねでマネしてほしい」
→ 「それでは片手を上に〜」

最後は観客と一緒に勝どきを上げる
「えいえいおー！」
→ えいえいおー！
「ももクロ、サイコー！」
→ いえーい！
→ 急に乗っかっちゃって大丈夫なんだろうかｗ

■第三部
12:30 本来のプログラムの予定演目であるトシ子ちゃんトークショー開始
こちらでもすぐに かなこちゃん登場
→ 呼びこむ前に出てきちゃった気がするｗ
→ 衣装は去年の桃神祭ももクロ衣装
→ だけど、トシ子ちゃん作ではないそうですｗ

かなこちゃんとトシ子ちゃんは、浜松出身つながり
か「ももクロは 5 人だとうわーといけるんですけど、ひとりずつだとそうでもないので」
か「最初はなんて話していいかとまどった」
か「けれど、浜松出身という共通点で近づくことができた」
→ 確かに緑の妖精さんは、得体がしれなくて近づきにくいよねー

地元トーク
→ このへんはまさに地元で、自転車で来られる距離だとか
か「地元のひとはザザとか、ジャスコとかに集まってる」

ももクロでの活動は、最初の頃は地元では全然まわりに話していなかった
か「部活のほうには、ダンスのレッスンで…」
か「ダンスには、部活で…」
→ みたいな感じに誤魔化していたそうです
か「でも、だんだんと辻褄が合わなくなって、嘘はいけないなと思いました」
か「そのうち、ネットとかで見つかるようになって、いきなり『えっ』みたいな感じになる」

トシ子ちゃんはももクロ衣装の中でカワイイ系衣装を担当
か「他にも色々な系統の衣装をつくるチームがある」
か「アーティスト路線の衣装では、顔がまったく出ていない衣装なんかもあった」
か「推され隊の 2 人の顔が全然でていなくて、今でもそのことを色々言われる」

ゆるキャラ出世大名家康くんもトークに参加していたんだけど、中のひとは結構トークがうまかった
→ 声の感じといい、ぶっちゃけ具合といい、ふなっしーぽい感じ
→ ゆるキャラグランプリの投票について組織票のお願い的なアピールがあった
→ 現在、僅差で 2 位につけているので今年はなんとかして 1 位をとりたい
か「モノノフのみなさん、投票してくれますかー？」
→ はーい！
→ がんばらないといけない雰囲気

「かなこちゃん、トシ子ちゃん、家康くん」
「レッツ出世だ〜」
→ ぜーっと！
で締めて退場 12:50

■第四部
ずっと立ちっぱなしでステージの張り込みをしていたので、少し休憩することに
13:20 フードコートエリアのテーブルでぐったりしていると、なんだか騒がしい
→ 別のステージにひとが集まっていたので行ってみると、かなこちゃんが出没してました
そちらでも家康くんと「ゆるキャラ城落とし合戦」イベントのゲストで色々と大活躍
油断していると神出鬼没な感じでのんびりしていられないｗ
13:32 退場

■のぶさま。
ゆるキャラのひとり
信長ちゃんモチーフの謎アニマル
警備員やスタッフの背後に近づき、手に持った小刀 (着ぐるみ的なモコモコ素材) でカンチョーしてましたｗ
→ きっと中には高城さんが入っているね

■第五部
16:00 からの合同閉会式に登場
司会者「危ないのでみなさん座ってください」
→ それまでスタンディングで結構な密度でしたが、大きな混乱もなく無事に全員着陸
→ きちんと指示に従っていた点をお褒めいただきました

司「浜松のアーティストのあの方をお呼びしましょう！」
→ おおおお？
「トシ子ちゃーん」
→ かなこちゃんも遂に「アーティスト」かと思ったらそうでもなかったｗ
その後すぐに登場 パレードの時の狩り衣装＋烏帽子

浜松市長の挨拶、トシ子ちゃんコメント、かなこちゃんコメントがあり餅撒き
か「こうやって地元で大勢のモノノフさんに会うことができて嬉しいです！」
最前に陣取っていたら、おこぼれをちょいちょいもらえました
→ ボーリングでいうところのガター処理的な余り？をゲット
16:20 餅撒き終了＆退場

閉会後、餅撒きのときにかかっていた BGM ワニとシャンプーのボリュームが大きくなり…
→ ちょうどサビだったので、みんなで立ち上がって踊って、とーぶーよっ！ジャンプしてきましたｗ
→ それを報道カメラマンがシャッターチャンスとばかりに撮影してましたｗ

■さわやか袋井本店
帰りがけに寄ることに
カーナビで検索したら、「有玉店」というグリーン＋イエローな感じのお店もあるようでしたが、せっかくなので ももたまい店へ
→ 桃神祭で ももたまいが社会科見学で訪れたお店
16:30 浜松城あたりを出発 17:20 着
ちょうど混み始める時間で、あとから続々と駐車場にクルマがやってきてた
30 分ほど待ってからテーブルへ
げんこつハンバーグのミックスソース＋ごはん大盛り
18:30 頃にお店を出て帰路へ
途中、海老名サービスエリアで れにちゃんを探すイベントをしつつ、おみやげの (かな) こっこを買って 22:30 鷺沼到着
