Animelo Summer Live 2014

さいたまスーパーアリーナへ行ってきたよ。

■けやき広場 ONENESS ステージ
開演前(14:00-)に たまアリの外のステージで TeddyLoid さんがももクロDJリミックスをやるというので早目に現地へ

13:00- アニソンカラオケ
事前に登録した出場者＋飛び込み？でステージでアニソンを熱唱
観客はこの段階で既にノリノリ 21:00まで体力が持つんだろうか？と心配になるレベル
アニオタといっても色々 派閥？があるようで、氣志團万博とは違う意味でカオス状態
飲酒可だったせいもあり、酔っ払いノリのひとが多かったような気がする

13:55- TeddyLoidさん登場 さっそくももクロDJリミックス開始
全部で60分ほどDJプレイをしていたけど、ももクロの曲は最初の10分くらいと最後の5分くらいのみ
残りはエヴァンゲリオン(残酷な天使のテーゼと魂のルフラン)とか、オリジナル曲？等

ノフさんはステージ近くで踊っているひとが100-200人くらいかな？
全体としては凄く少な目 今日は1割以下だったと思う
7月からの各現場と比べると、男性比率高め

■天気
午前雨、午後くもりといった予報で、午後は雨が降らずに14:00過ぎからは日が差してきてむしろ暑いくらい

■会場・座席・音響
今日はスタジアムモードかな？ 観客 27,000人 500レベルまできっちり埋まってた
自分の席は上手側200レベル、サブステージより少し後方といったところ
音響は悪くなかったと思うけど、後半の方が音が大きくて良かった印象
観客がジャンプしていることが多く、自分のいるエリアは常に震度2といった具合 ずっと揺れてた
→ ももクリ2011のオープニングでカメラが揺れてたのを思い出した
開演前などは nirvana の smells like teen spirit や metallica の enter sandman が爆音で流れるなど、ロック寄りの選曲
Nirvana - Smells Like Teen Spirit

■最後の挨拶
各グループごとに挨拶をするところで、ももクロちゃん代表として元リーダー高城さんがコメント
れ「来年も出られるように、アニメロもじゃんじゃん歌いたいと思います！」
れ「堀江師匠ともコラボできました！」
→ 堀江さんと、隣にいたももかちゃんがツーショットでスクリーンにアップで映し出される
→ 2人でほっぺたを寄せ合ってすごく楽しそう＆ももかわいい (強調)
天井からハート型の紙ふぶきが紙飛行機みたいにちらほら舞ってきていて、ももかちゃんがそれを1つ空中でキャッチしてた
→ 他のメンバーもそれぞれゲットして手に持ってた

最後に出演者全員でサブステージのほうまで練り歩き、観客に挨拶をして退場
この時の衣装がミニスカートで、脚の露出が高かったんだけど、あらためて ももかちゃんの脚がほっそりしてきてるなーと思った
21:20に終演 16:00から5時間半近く、ほぼずっと立ちっぱなしでした
