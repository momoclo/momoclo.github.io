京セラドーム大阪 Day 1

京セラドーム大阪に行ってきたよ。
メモは避難所 239 参照。まだまだ演出は伏せ気味にレポートします。

■会場・客入り
京セラドーム大阪 大阪には何度か来たことがあるけれど、ドームは初めて来ました
→ 学生の頃は夏休みや冬休みに京都のおじいちゃん・おばあちゃんの家に1ヶ月単位で居候してた
ナゴヤドームよりもお客さんが多い印象 スタンドは最上段あたりの特別席？っぽいところ以外は埋まってた
ドームすぐ横のイオンでは、ナゴヤドーム同様に ももクロを意識した販促コーナーがたくさんｗ
→ 5 色並びのお菓子やドリンク売り場などなど

■聖地巡礼
アメリカ村 三角公園 → 猛烈宇宙交響曲・第七楽章「無限の愛」のリリースライブをやったらしい？
一鶴 心斎橋店 → Chan の香川 ぶらり高城・こってりパトロールで食べてた骨付き鳥のお店
心斎橋 だるま → 大阪城ホールの女祭りの時に寄った串カツ屋さん
他どこかあったら教えてください (・Θ・)

■ジャンプ禁止
京セラドーム大阪は観客がジャンプをすると周辺住民の迷惑なので禁止とのこと
→ きちんとルールが徹底されていて、ジャンプをしているひとは見かけなかった
→ でも Chai Maxx とかは少しスタンドが揺れてました

■タイムテーブル
18:00 頃 - 19:30 一部
19:30 - 20:50 二部
20:55 - 21:20 アンコール

■今日の有安さん
一時期よくやっていた、ばきゅーんが復活
→ 大阪だからなのかな？
ステージからスタンドの上のほうとか、結構長距離もばきゅんばきゅんしてました
髪型は編みこんだ感じのサイドポニーテール

ちょっとこのスレッドで話題になっているけれど、花粉症で少しつらそうな感じ？
→ 自己紹介では…
も「みなしゃん、てびょびょ」 ← カミカミｗ
も「かふんちゅ…、かふんちゅう…、かふんしょうちゅうの有安杏果です！」 ← カミカミｗ
確かに緑推しが注意して見ていれば、花粉症で歌いにくそうな感じの時もあったけれど心配するほどではないと思いました

ボールを観客席に投げ込むシーンでは、やっぱり緑推しをロックオンして投げ込んでいる様子
→ スタンドから双眼鏡でボールの行き先を確認しましたが、結構な割合で緑推しがゲットしてました
→ といっても、コントロールがイマイチなので緑推し以外がゲットすることも当然あり

■MC
フリーダムすぎてまとめるのが難しい…

■アゴでフラフープ
ちょっと失敗してフラフープがあさっての方向に飛ぶハプニング
→ ステージから転がって落ちちゃうんじゃないかとヒヤリとしましたが、うまいことリカバー
「大阪なので、失敗したと見せかける高度なテクニック」 ← 的な言い訳をしてましたｗ

■ポケモンシステム
かなこちゃんが (ネタバレ) を覚えた！
→ 同時に 3 つまでしか覚えられないので、代わりに何かを忘れるポケモンシステムが発動ｗ
あ「『否めない』の使い方とか忘れてるんじゃない？」

このあたりのシステムについては、前にどこかの MC でもあったので覚えているひとや、ポケモンに詳しいひとならわかると思うけれど…
も「みんなぽかーん状態なんじゃない？」

■手汗
しおりんとれにちゃんがハイタッチをした時に…
し「うわー、れにちゃんの手汗がべちょべちょだあ〜」
→ ここから、こちらも懐かしの安藤なつさんの話題となり…
メイプル超合金とか、かなこちゃんが「あんドーナツ」の安藤なつさんと思っていた？などのディープな用語・話題が飛び交う状態にｗ
も「みんなぽかーん状態なんじゃない？ (・Θ・)」
→ 天丼ですね、わかります

■アゴでギター
ナゴヤドーム・札幌ドームよりも小技が増えてきました
→ 弦をかきならしながら 1 → 20 フレットくらいまで音程を上げたり
→ ネックのヘッド部分、ペグのあたりをかきならして音を出したりなどなど

■アゴでドラム
フィルインのあとにスティックが飛んで行っちゃってたよね？
→ ストックのスティックをすかさず出していたので演奏には支障なし

■大阪
MC のウケがよいかどうか、大阪のひとに緊急アンケートしてみよう in 京セラドームが開催
れ「今のトーク、どうでしたか？」
「あんまり面白くなかったってひとー？」
→ 結構な人数が挙手してましたｗ
いきなりズバリで訊いてみるのもアレですが、正直な反応が返ってくる大阪は手強いですねｗ
「ま、まだ後半に巻き返そう！」

→ 今日の MC はポケモンシステムとか、手汗ネタとか、難易度が高めだった気がしますｗ

■黒い週末
れ「ここで、みなさんにお知らせが…」
→ やめないでー！
→ と、いつものお約束の流れだったんだけど、自分の後ろの れにちゃん推しの女性が「今までありがとー！」と既にやめちゃっているかのような発言をしてたのがおかしかったですｗ

れ「実は、高城も花粉症でした…」
れ「去年、血液検査をしたら 98% の割合？で花粉症で、この時期は大変で…」
→ ごほっごほっごほっ…からの〜イントロ

■行くぜっ！怪盗少女
最後のフェイクパートからの戻りはセーフ

■れにちゃんソロコンサート
先日のソロコンサートでも話していた内容を今日も話してました (どういう流れでだったかは忘れた)
れ「小学 6 年生のときに、将来の夢で」
れ「タップダンスを踊っていたい、というのと」
れ「歌手になりたい、という 2 つを書いたんだけど、10 年後の今、その夢が叶ったよー！」
→ おめでとうー！

か「10 年前の高城さんに、今の自分から何か言うことは？」
れ「10 年前は、ママのおもちゃだったけど、そのまま まっすぐ育ってね」 ← 内容うろ覚え
→ とにかく、夢の実現とはあんまり関係ないことを言っており、その点についてメンバーからダメ出しをされてましたｗ

■ももたまい婚
玉井さんが嬉々として大発表
→ スクリーンに「ももたまい婚」と出てましたｗ
→ そして熱いハグを交わす ももたまいのお二人
「せっかくだから、二人の共同作業でケーキ入刀とかやったら？」
→ ウェディングプランナーの意見をノリノリで聞く玉井さんと、あんまり乗り気じゃなさそうな百田さん

きちんと確認していないけれど、9/4 (日) 新潟県民会館だっけ？
→ キャパシティが 1500-1700 人らしく、飲み会では赤推し、黄推しさんが倍率の高さに慄いてました

■ももクラシック的な何か
ドームツアーの映像を振り返る大打ち上げ会のようなイベント
→ こちらも公式発表をあまりよく確認せずに書いています もう眠い…
4/29 (祝) 東京ドームシティホールだっけ？
ライブではないようなので、武道館の ももクラシックっぽい感じなのかな？

■ジャンプ禁止
大事なことなので 2 回…ではなく、最後の最後に
か「最後にみんなで飛ぶよー！」
→ あちこちでペンライトでバツを作るひとがいた気がしますｗ
じゃかじゃかじゃか…じゃん！
→ で ももクロちゃんは飛んでましたが、観客は ほとんど飛んでなかった
→ みんなきっちりルールを守っていたよね

壮大な釣りかと思いましたが、本当にやらかしちゃったらしくバックステージにはける前くらいにコメント
か「さっき、ジャンプ禁止なのに飛ぶよー！って言っちゃった」
か「もしかしたら一回だけ、ちょっと足が地面から離れちゃったかも…って関係者に謝っておきます！」
→ ちゃんとドームさんに謝っとこうよ、ということで…
「どうもすみませんでした」
→ MC の最後に謝罪をする妙な空気にｗ
→ きちんと謝ったので、スッキリと明日のライブに臨めるね、という具合に前向きに締めて終了

■まとめ
知り合いの関西ノフさんに今日の様子を聞いたら、一般席の周りはノリが関西ぽかった (新規？の方などがよく知らなくてもとりあえず盛り上げとけみたいな感じになるとか) と言ってました。
ももクロちゃんのコメントでも「やっぱり京セラドームだとちょっと雰囲気が違う」と言っていたので、そういうのはわかるもんなのかなーと思いました。
