俺の藤井 2016

俺の藤井 2016 Tynamite!! に行ってきたよ。
MC がなかったので、軽めにレポートします。

■会場
さいたまスーパーアリーナ
今日はアリーナモードかな？ 10,000 人くらい
アリーナもすべて指定席で、自分はアリーナ D6 ブロック
→ アリーナの一番後ろのブロックでしたが、結構見やすかったです

■こってりパトロール
SSA に行く前に大崎の平太周へ
15:30 頃に到着したら、満席でした
→ 15:00 から販売開始の爆盛油脂麺を注文するひとで混んでいた模様
自分は りんりんコンビと同じメニューということで、特製ラーメン＋味玉を注文
トイレのドアにパトロール隊のステッカーが貼ってあり、その上にパトロール隊サイン＋隊員見習い山里さんのサインが飾ってあった

■開演前
17:30 くらいに入場したけれど、BGM なんだったっけ…
→ まったく覚えていませんが、ひょっとしてかかってなかった？
17:55 ワンデイワールドリーグ戦のルール説明のビデオが流れる
→ ナレーターは太田真一郎さんかな？
https://youtu.be/DS98NyLu58I?t=7m
ルールはラストマン・スタンディングならぬラストアイドル・スタンディング方式
→ 雌雄は観客の声援の大きさで決める
→ それぞれの出番の後に 5 秒ずつ計測タイム
→ すぐ後にデシベル表示 (103.5 dBとか) でデジタルに勝敗を判定

■開演
18:00 スタート
まずは全出演者が勢揃いして顔見せ
各グループの煽りムービーが流れましたが、ももクロちゃんは有安さんメイン
→ 有安さん・邪道さん・誰か の 3 人で「ももクロが絶対王者だー！」といった内容
ももクロちゃんは出番までしばらくバックステージへ

■実況・解説・レポーター
実況 清野さん
解説 Quick Japan コジマさん

バックステージ オテンキ のりさん
→ バックステージでは ももクロちゃんとの絡みが多かった
→ れにちゃんが のりさんのジャージズボンを脱がそうと引っ張ってたよねｗ
→ 小学生か！と思いましたｗ

楽屋 アナログタロウさん
→ アナログさんは今日も豆知識を披露してましたｗ
ア「ムニャムニャさんが一番よく使う乾電池は、単3だそうです」
ア「ホニャララさんがイカフライを食べたら、最初の一口で中身が全部出ちゃったそうです」

■ももクロちゃん登場
19:15 過ぎくらいに登場、対戦相手は私立恵比寿中学
いきなり 行くぜっ！怪盗少女

■衣装・髪型
桃神祭の衣装を少しミニスカートにした感じ？
遠くで現場ではよくわからなかったけど、あーりんが髪を切ったらしい
あ「この日のために髪を切ってきました！」
→ と気合の入り具合をアピール

■行くぜっ！怪盗少女
後半フェイクパートはまず下手側へ
→ その後上手側へ
→ どちらもステージの端まで行って、そこでも少し止まっていたけれど最後の決めポーズには余裕で間に合ってました

■灰とダイヤモンド
砂にまかれてーもー の気張りやすパートはオケ通りの長さだったので、特に引っ張ったりビブラートをかけている感じではなかった
→ 今日一番、よく声が通っていたと思います
実況の清野さんに「バラードという寝技で攻めてきました」とか言われてましたｗ

■MOON PRIDE
ここまで連勝中でしたが、チームしゃちほことの対戦で敗退

■ツヨクツヨク
タオルは各グループのものをメンバーそれぞれが持つ形
→ ももかちゃんは水色っぽいのを持っていたっけ？
→ 今日もだいぶもたついていましたが、無事に上下正しく出してましたｗ
この曲の途中で Final Round となり、チームしゃちほこに勝利して優勝

■Chai Maxx
全出演者で Chai Maxx
緑の衣装のひとがいっぱいいてちょっとわかりにくかったですｗ

■勝利コメント
れ「今日は戦っているけれど、みんな仲間なので〜」
→ スターダストアイドル同士のバトルということで、ギスギスした感じになるかと思っていましたがいいコメントでした

あ「中には、『なんでももクロが優勝なの？』と思うひともいるかもしれないけど…』
あ「ていうか、みんなスタダDDでしょ？」
→ はーいｗ
あ「で、みんなあーりんのファンだと思っているから」
→ あーりんキャラは強いなーと思いましたｗ

し「一年に一度こういうイベントで、後輩グループのみんなが成長しているのがよくわかります」
し「先輩として、恥をかかないように頑張っていきます」

も「今日、お休みしちゃったメンバーもいるんだけど…」
→ 私立恵比寿中学とチームしゃちほこでお休みしたメンバーのことを気遣ってました
も「自分は去年、インフルエンザで出ることができませんでしたが、今年は出ることができました！」
も「今日、初めて見たグループもこれをキッカケに応援してね」
→ それぞれのファンに対して、初見のグループについても宣伝してました

か「今日は楽屋で色々なチームが本気で練習していたので」
か「ももクロも団結が凄かった」
→ 負けないように気合を入れていたみたいですねー
か「これからもみんなを引っ張っていけるように、背中を見せていきたいです！」
→ すかさず、しおりんが…
し「こっち？」
→ と、裏表を間違えてましたｗ
か「こっち！」
→ 背中を指してアピールｗ

最後に理事長から大きなチャンピオントロフィーと明日の大トリ出演権目録を授与されて終演 21:03
→ CHAMPION の綴りが間違ってましたｗ
→ CHANPION になってたよね

■まとめ
最後に全員そろった時に、やっぱり後輩たちからの視線が頼りにされているなあという感じでした。
れにちゃんのことをスカートめくりして喜ぶ後輩がいたり、年明け早々に
か「2015 年もよろしくお願いします」
と言っちゃうポンコツリーダーだけど、スターダストのアイドルを引っ張るのはやっぱりももクロちゃんであり、本当にみんなから慕われているなと改めて思いました。
