ハスラー 富士急ハイランド

富士急ハイランドに行ってきたよ。

■ブロック・観客
一般 D2 ステージ中央よりも少し下手 すぐ前がオーナー席
一般ブロックは全部うまってました
オーナー席は 7 割くらいかな？
4 人の家族連れで、20 歳前後のこどもが興味があって家族で見に来ましたという感じの方々もチラホラ
→ 50-60 歳くらいのお父さんが一般ブロックを不思議そうに見てましたｗ

■お天気
前日の夜に天気予報を確認したときは、降水確率 100% 6mm/h
当日朝、渋谷のバスターミナルに着いた時は雨
…だったんだけど、高速道路を西に行くにつれて雨がやみ、富士急ハイランドは地面が少し乾いてるくらい降ってなかった
雨雲レーダーを見たら、雨雲が富士急ハイランドの南に追いやられてましたｗ
結局、待機中〜開演〜終演までずっと雨具要らず
終演のときに少し雨が降ってきて、帰りのバスを待つ間だけ雨模様だったけれど 17 時過ぎにはまた止んでました

■開演
14:05 Overture スタート

■髪型
おさげっぽい感じのツインテール

■ニッポン万歳

→ まわりの緑推しのテンションが明らかに上がってたｗ

■ツヨクツヨク
最後のタオルはよ〜く確認した結果、逆さまじゃありませんでした

■あーりん先生
ハスラーについて、あーりん先生がハスラーロボの格好で解説

「ハスラー、ピンクが一番人気らしいよ」
→ 何度かピンクというワードが出てくる会話をする横で、ハスラーロボーりん先生が「ピンク？」のボケで絡むことができずにメンバーをじっと見てましたｗ
→「ピンチ！」に「ピンク？」で会話に混ざることができても、「ピンク」はヒネリが無さ過ぎて会話に混ざってこられない模様

あ「さて、では」← 明美ちゃん先生風
ハスラーの機能について生徒に質問して答えさせるんだけど、呼び名が「ちょいちょい悪口はいってる」
れにちゃん → 妖怪
かなこちゃん → 浜松っこ
しおりん → 茶髪くん
ももかちゃん → アマガエルくん

あ「さて、では、ムニャムニャ (略) という機能のことを、なんというでしょう？はい、アマガエルくん！」
も「S-エネチャージ！」
あ「いつもネットばかり見てる割りによくわかってますね」
も「ハイ！」← 恒例？のかなこちゃんが元気よく返事するのと同じ感じで元気よく返事してた

■Los Angels
MC が何度かあったけれど、ちょくちょく LA の話題が登場
れ「USJ にも行きました！」
「J じゃないよ、US…H だよ」
「ユニバーサル スタジオ ハリウッドじゃないとわかんないよ」
→ クルマで園内のあちこちにある映画のセットを見て回ったりしたそうです

普段、滅多に雨が降らない LA で、ももクロちゃんが到着したらいきなり雨
「ツアーのコーディネーターさんにも驚かれた」
か「あっちは一年 365 日中、300 日しか雨が降らないらしいんですよ！」
→ 色々まちがっていてツッコミどころ多数ｗ

れ「LA から帰ってきたと思ったら、すぐに富士急だよー！」
→ タイトスケジュールについて言いたいことがあるようです
れ「芸能界は、嘘ばっかりだー！」
→ 本来は れにちゃんに続けて残りのメンバーが発言を繰り返す流れだったんだけど、このセリフには誰も乗ってこなかったｗ
も「いやー、スズキさんステキなライブを用意してくださってありがとうございます♪」
→ さすが芸歴 20 年、芸能界の渡り方をよく心得ている（・Θ・）さん

レストランで食事中、店員さんがお皿をかなこちゃんのアタマにぶつけちゃう事件発生
→ 店員さんが英語で「すみません、大丈夫ですか？」らしきことを言っていたそうですが…
か「イエーイ、サンキュー！」
その後も英語で「さきほどはどうもすみませんでした」らしきことを言われて…
か「ユーアーウェルカム！」
→ 本人は「ノープロブレム」と言いたかったらしいｗ
→ 「ノープロブレム？」「ノープログレム？」みたいな感じでメンバー全員かなり怪しい英語力を披露してました
No Problems なので「ブ」ですねー

オーナー席の 4 人家族のお父さん、ライブ中はスタンディングでただ見ているという感じでしたが…

→ 単純に合わせやすいというのはあると思うけれど、アタマの上で手拍子してくれてました

■富士急ハイランド
開演前に、ももかちゃん、かなこちゃん、しおりんの 3 人で FUJIYAMA に乗ってきたそうです
か「両手をバンザイしていると、落ちるときに首が後ろに持っていかれちゃう」
か「けど、今日そうならないコツがわかった！」
→ アゴをひいて、ダイブするような形で前のめり気味にするのがポイントらしいです

■亜土ちゃんとお絵描き
「今日もステキなゲストの方がきてくださっています！」
→ どこー？
か「あ、見てー！」
→ 客席の後方を指さす百田さん
→ つられて後ろをキョロキョロ探していると…
か「バカが見る〜♪」
毎年、夏の野外でだまされるのが恒例になりつつあるのかなｗ

あらためて水森亜土さんがメインステージにハスラーで乗り付けて登場
夏ということで、泳ぐ ガッちゃん (亜土ちゃん担当) と、海の生きもの＋その他色々 (ももクロちゃん担当) をアクリルボードに両手描き
→ 亜土ちゃん、初めて見ましたが両手描きだけじゃなくて、さらに歌いながら絵を描くんですねー

有安画伯の作品解説
おさかな (最初 水平線よりも上に描いていたと思ったので、カラスかと思いましたｗ)
→ その後、上にも水平線が出現して空の生きものから海の生きものに変貌
イカ (四角いブロックに棒が生えている謎物体)
カニ (脚が 4 本くらいだけれど、カニに見える)
ヨット (波の上に浮いた何かなので、たぶん船だよね…)

絵を描くときの仕草がモニターに映っていたけど、なんだかやけに真剣そうに、右と左の手の先をキョロキョロ首を振りつつ善戦してましたｗ

し「お絵描きスキルがないと (亜土ちゃんと比べられちゃって) 大変だよー」
あ「しおりんは絵がうまいほうだからマシだよ」
（・Θ・）「あたしなんて、みんな知っている通りだから つらいよ…」

ハスラーにもお絵描きしよう
左右のドアは北九州、姫路で既に絵を描いてあるので、今回はボンネットに
亜土ちゃんが ガッちゃんを描いて、しおりん、あーりん (ピンクは亜土ちゃんが塗ったっけ？) がカラースプレーで彩色
グリーンも足してみようということで、亜土先生からリズミカルに色を塗るように指示が出される
→ 一度、先生にお手本を見せてもらってから有安さんも彩色してました
次にレッドも、ということで百田さんが同様に…
→ ちょっとスプレーの量が多すぎて事故っぽい感じになってましたｗ
→ その時は気がつかなかったけれど、後で「かなこちゃんは左利きなのに、マイクをみんなと同じように左手で持つから、スプレーは利き手じゃない右手で持ってた」と聞きました
最後にパープルで完成
亜「紫はちょっとだけね」
→ オチ担当っぽい扱いｗ

■灰とダイヤモンド
か「今日も一番いいところで気張りやすのコーナーです！」
ハスラー＋サクミさん＋重り、もとい大森さんの合計 2 トンを有安さんが綱引きみたいなロープで引っ張る
→ 今日は本当に重かったらしく、れにちゃん以外の 4 人で S-エネチャージよろしく力を合わせて引っ張ってました
→ そんな様子を反対側から見守るれにちゃん

■ワイワイワールド〜終演
終盤の MC でお天気の話題に
「なんとか雨が降らずにもったね」
「少し空が暗くなってきてるけど、夕方になってきたからだよね」
あ「雨ふれー！」
「そんなこと言ったら本当に降ってきちゃうよ」

最後にふたたび亜土ちゃんがステージに登場、一緒にワイワイワールドをみんなで合唱 (スクリーンに歌詞が出てた)
歌詞が出ていたのでわかったけれど、ももかちゃんがソロパートで歌詞を少し間違えてたよね
→ 歌詞サイトを確認しつつ、思い出すと「笑いのバクダン (3番)」のところを「夢のバクダン (1番)」と歌っていた気がする
→ 歌いながらハスラーに各自乗り込み、退場していったんだけど…
→ このくらいから雨がポツリポツリ
ももクロちゃんたちが退場したら、小雨になりました
15:50 終演

■まとめ
月刊 TAKAHASHI は今のところ全部 LV なので、ライブの現場は どんたく以来かな？
ももクロちゃんお天気伝説ぶりも相変わらず凄く、心配していた雨も降らず、かといってクノのような猛暑にもならず、野外でフリコピするにはちょうどいいお天気でした。
セットリストも良かったし、久しぶりで楽しかったです。
楽しいステージを準備してくれた SUZUKI さん、ハードスケジュールにもかかわらずいつも同様に元気をくれた ももクロちゃんありがとう。
