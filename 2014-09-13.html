氣志團万博 2014

氣志團万博に行ってきたよ。

帰ってくるまでに時間がかかった分、帰りの電車内でTwitter TLやらまとめを読んで色々と思い出して長文になったけど、まとめだと思ってもらえれば。

■会場・音響・天気・物販
去年と同じく袖ヶ浦海浜公園
観客数は 25,000 人弱とのこと (by 氣志團あやのこぉ〜↑↑さん MC)
モノノフ率はだいぶ高かったと思う 7 割くらいかな？

今年は雨対策として、スタンディングエリアのダート部分には芝生保護パネルみたいなのがひいてあり、ぬかるみ状態にならない工夫がされてた
→ つい2-3日前に大雨が降ったので、少し心配してたけど全然問題なし

ももクロちゃんの時は最前2列目だったので、音響はとてもよかった
特にベース音は肺に振動が来る感じ

自分は11:20分頃に会場到着 ちょうどニューロティカさんが始まってすぐ
思ったよりも長浦から時間がかかって、もりえもんのオープニングアクト見られなかった…
氣志團万博は去年に引き続き2回目の参加
→ その後、前のほうだったり、最後尾だったりとバラバラだけど結局 全部通しで見てきた
→ 長くなるので他のアーティストについては ももクロちゃん関連以外は省略
→ 私立恵比寿中学もよく知らないなりにレポート書けそうだけど長くなるので(以下同文)
→ もりえもんは途中、きゃりーぱみゅぱみゅさんの前くらいのインターバルの時に再登場
も「オープニングアクトも無事終わって、海を見て黄昏ていたら、翔やんに時間が余ってるからなんかやってと言われましたｗ」
→ 「どこもかしこも駐車場」という曲を歌った
http://youtu.be/tOLsZr7Dt5k

14:00頃、ほんの少しだけ小雨がぱらついたけど天気は曇り時々晴れ
小雨の時、虹が出てた
→ 日産スタジアム2日目を思い出した
暑くもなく、寒くもなくよいお天気 去年よりは涼しく感じた
日中でも日蔭ならかなり快適

物販はももクロちゃん関連は午前中くらいで売り切れたのかな？
係りのひとにきいたら13時には無かったとのこと

■労働賛歌 by 筋肉少女帯
ケンちゃん「ちょっと待ってね、この曲はイヤモニしないと歌えないの」
→ それまでイヤーモニターをしていなかったけど、おもむろに服の襟からイヤモニを取り出し装着ｗ

→ そしてその光景を見た筋肉少女帯のメンバーが全員驚いてたｗ

■行くぜっ！怪盗少女
去年は かなこちゃんの前転6連発があったけど、同じパートで…
→ 今年もボクシングのラウンドインターバルみたいに椅子が出てきて かなこちゃんを座らせ、ビール瓶でお水をぐいっと
→ その後、前転をするかと思いきやサブステージには大きな座布団みたいなマットが登場
→ そのマットで逆立ちをして三転倒立！
→ その状態のまま、マイクを高橋名人 (だったよね？) が口のところに持っていきソロパートを熱唱！
→ さらに逆立ち状態のまま、開脚バランスまで披露

終盤ももかちゃんフェイクパートではまずステージ下手へ
→ キター！とばかりに狂喜する下手側 真緑会員 (注：そんなものは無い) のみなさまｗ
→ その後ステージ上手にダッシュ！
→ 上手も端っこまで行って、曲の終わりには とてとてとてっと走り切って間に合った
→ 最後、段差をちょっとジャンプして乗っかってたよね

■コノウタ
あんまりモニターには映っていなかったと思う (カメラは追っかけてきていたので録画しているはず) けど、曲全体の半分くらいは有安さんがステージ下手に来て歌ってくれた
→ キター！とばかりに狂喜する (以下同文)
→ 有安さんもとっても嬉しそう
→ この時は軽く圧縮がかかってたｗ
→ ステージの端っこのさらに端、もう落っこちちゃうんじゃないかというくらいギリギリまで端に寄って、スタンディングエリア左端のひとにも手を振ってくれたり、レスをしていた

有安さんのソロパートで、「自分の限界」のところが凄く気合いが入っていたように感じた

分かれ道で行き先 悩んだ時
自分の限界に 立ち止まってしまった時

→ 最近だと、めざましライブの「宮城の牛タンだいすきです！」の歌い方とか、古くだと2011女祭りのコノウタの同じパートみたいな気迫を感じた (調子が悪くてみんなにバックアップしてもらいつつ、負けじと存在感を出していたアレ)

コノウタが終わって、ぜーっと！の挨拶をして退場 (20:10)

■明日以降行くひと向け
今年は去年と違って、長浦からのバス到着位置が会場から離れてます
→ 去年、帰りのバス乗り場になった砂利広場に到着
なので、会場まで少し時間がかかるかも

終演時間は少し早まったので、終電には間に合いそう
→ 去年は 21:00 過ぎ終演で、バスに乗ったのが 23:00 前、袖ヶ浦発の終電 (23:15頃) にギリギリ間に合った
→ 今年は 20:30 終演で、そのままひとの流れに乗ってバスに乗ったのが 22:00 駅についたのが 22:15 頃
